# <a name="pidtaginitials-canonical-property"></a>Kanonische PidTagInitials-Eigenschaft

  
  
**Betrifft**: Outlook 
  
Enthält die Initialen für Teile des vollständigen Namens des Empfängers. 
  
|||
|:-----|:-----|
|Zugeordneten Eigenschaften:  <br/> |PR_INITIALS, PR_INITIALS_A, PR_INITIALS_W  <br/> |
|Bezeichner:  <br/> |0x3A0A  <br/> |
|Datentyp:  <br/> |PT_UNICODE PT_STRING8  <br/> |
|Bereich:  <br/> |Adresse  <br/> |
   
## <a name="remarks"></a>Hinweise

Diese Eigenschaften Identitätsnachweis und Zugriff auf Informationen für einen Empfänger. Sie sind durch den Empfänger und ihre Organisation definiert.
