# <a name="pidlidfax3addresstype-canonical-property"></a>Kanonische PidLidFax3AddressType-Eigenschaft

  
  
**Betrifft**: Outlook 
  
Gibt den Adresstyp für die anderen Fax-Adresse des Kontakts an.
  
|||
|:-----|:-----|
|Zugeordneten Eigenschaften:  <br/> |dispidFax3AddrType  <br/> |
|-Eigenschaft festgelegt:  <br/> |PSETID_Address  <br/> |
|Long-ID (Abdeckung):  <br/> |0x000080D2  <br/> |
|Datentyp:  <br/> |PT_UNICODE  <br/> |
|Bereich:  <br/> |Kontakt  <br/> |
   
## <a name="remarks"></a>Hinweise

Diese Eigenschaft muss auf "FAX" Falls vorhanden, festgelegt werden.
