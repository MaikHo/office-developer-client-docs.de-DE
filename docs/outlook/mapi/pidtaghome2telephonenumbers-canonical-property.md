# <a name="pidtaghome2telephonenumbers-canonical-property"></a>Kanonische PidTagHome2TelephoneNumbers-Eigenschaft

  
  
**Betrifft**: Outlook 
  
Enthält eine Liste der sekundären Telefonnummern zu Hause des Empfängers.
  
|||
|:-----|:-----|
|Zugeordneten Eigenschaften:  <br/> |PR_HOME2_TELEPHONE_NUMBER_A_MV  <br/> |
|Bezeichner:  <br/> |0x3A2F  <br/> |
|Datentyp:  <br/> |PT_MV_UNICODE8  <br/> |
|Bereich:  <br/> |MAPI-e-Mail-Benutzer  <br/> |
