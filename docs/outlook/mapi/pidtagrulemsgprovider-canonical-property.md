# <a name="pidtagrulemsgprovider-canonical-property"></a>PidTagRuleMsgProvider (kanonische Eigenschaft)

  
  
**Betrifft**: Outlook 
  
Enthält zusätzliche Informationen über die Clientanwendung, die eine erweiterte Regel besitzt.
  
|||
|:-----|:-----|
|Zugeordneten Eigenschaften:  <br/> |PR_RULE_MSG_PROVIDER  <br/> |
|Bezeichner:  <br/> |0x65EB  <br/> |
|Datentyp:  <br/> |PT_UNICODE  <br/> |
|Bereich:  <br/> |Regeln  <br/> |
