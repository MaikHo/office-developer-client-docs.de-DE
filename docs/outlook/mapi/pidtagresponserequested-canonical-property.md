# <a name="pidtagresponserequested-canonical-property"></a>Kanonische Pidtagresponserequested (-Eigenschaft

  
  
**Gilt für**: Outlook 2013 | Outlook 2016 
  
Enthält TRUE, wenn der Absender der Nachricht eine Antwort auf eine Besprechungsanfrage wünscht.
  
|||
|:-----|:-----|
|Zugeordnete Eigenschaften:  <br/> |PR_RESPONSE_REQUESTED  <br/> |
|Kennung:  <br/> |0x0063  <br/> |
|Datentyp:  <br/> |PT_BOOLEAN  <br/> |
|Bereich:  <br/> |MAPI-Umschlag  <br/> |
   
## <a name="remarks"></a>Bemerkungen

Diese Eigenschaft wird für Besprechungsanfragen verwendet. Die empfangende Clientanwendung sollte den Benutzer auffordern, die Anforderung zu akzeptieren oder abzulehnen und diese Antwort dann an den Absender zurückzusenden.
