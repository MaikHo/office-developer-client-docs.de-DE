# <a name="pidlidfax3emailtype-canonical-property"></a>PidLidFax3EmailType (kanonische Eigenschaft)

  
  
**Gilt für**: Outlook 2013 | Outlook 2016 
  
Gibt an, dass der e-Mail-Typ des Kontakts Faxadresse zugrunde.
  
|||
|:-----|:-----|
|Zugeordnete Eigenschaften:  <br/> |dispidFax3EmailType  <br/> |
|-Eigenschaft festgelegt:  <br/> |PSETID_Address  <br/> |
|Long-ID (Abdeckung):  <br/> |0x000080D7  <br/> |
|Datentyp:  <br/> |PT_UNICODE  <br/> |
|Bereich:  <br/> |Kontakt  <br/> |
