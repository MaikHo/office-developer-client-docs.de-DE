# <a name="pidlidfax2richtextformat-canonical-property"></a>Kanonische PidLidFax2RichTextFormat-Eigenschaft

  
  
**Betrifft**: Outlook 
  
Gibt das Format rich-Text, der die private Adresse des Kontakts Fax an. 
  
|||
|:-----|:-----|
|Zugeordneten Eigenschaften:  <br/> |dispidFax2RTF  <br/> |
|-Eigenschaft festgelegt:  <br/> |PSETID_Address  <br/> |
|Long-ID (Abdeckung):  <br/> |0x000080C6  <br/> |
|Datentyp:  <br/> |PT_BOOLEAN  <br/> |
|Bereich:  <br/> |Kontakt  <br/> |
