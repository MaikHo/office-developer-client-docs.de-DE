# <a name="pidlidcontactuserfield3-canonical-property"></a>Kanonische Pidlidcontactuserfield3 (-Eigenschaft

  
  
**Gilt für**: Outlook 2013 | Outlook 2016 
  
Gibt das dritte Feld des Kontakts an, das für verschiedene Zwecke für den Kontakt vorgesehen ist.
  
|||
|:-----|:-----|
|Zugeordnete Eigenschaften:  <br/> |dispidContactUserField3  <br/> |
|Eigenschaftensatz:  <br/> |PSETID_Address  <br/> |
|Long-ID (Deckel):  <br/> |0x00008051  <br/> |
|Datentyp:  <br/> |PT_UNICODE  <br/> |
|Bereich:  <br/> |Kontakt  <br/> |
   
## <a name="remarks"></a>Bemerkungen

Diese Eigenschaft enthält Text, der keine Beziehung zu einer anderen Kontakt spezifischen Eigenschaft hat.
