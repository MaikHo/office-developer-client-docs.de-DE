# <a name="pidnamexsharinglocaltype-canonical-property"></a>PidNameXSharingLocalType (kanonische Eigenschaft)

  
  
**Betrifft**: Outlook 
  
Stellt die Zeichenfolgenversion der **DispidSharingLocalType** ([PidLidSharingLocalType](pidlidsharinglocaltype-canonical-property.md))-Eigenschaft.
  
|||
|:-----|:-----|
|Anzeigenamen:  <br/> |Keine  <br/> |
|-Eigenschaft festgelegt:  <br/> |PS_INTERNET_HEADERS  <br/> |
|Name der Eigenschaft:  <br/> |X-Freigabe-Local-Typ  <br/> |
|Datentyp:  <br/> |PT_UNICODE  <br/> |
|Bereich:  <br/> |Freigabe  <br/> |
