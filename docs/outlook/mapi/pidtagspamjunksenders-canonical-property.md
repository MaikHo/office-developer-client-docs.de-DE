# <a name="pidtagspamjunksenders-canonical-property"></a>Kanonische Pidtagspamjunksenders (-Eigenschaft

  
  
**Gilt für**: Outlook 2013 | Outlook 2016 
  
Enthält eine durch Semikolons getrennte Liste von e-Mail-Adressen und Domänen, die blockierte Absender enthalten.
  
|||
|:-----|:-----|
|Zugeordnete Eigenschaften:  <br/> |PR_SPAM_JUNK_SENDERS_W  <br/> |
|Long-ID (Deckel):  <br/> |0x041A  <br/> |
|Datentyp:  <br/> |PT_UNICODE  <br/> |
|Bereich:  <br/> |Spam  <br/> |
