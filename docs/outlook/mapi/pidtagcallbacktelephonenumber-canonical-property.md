# <a name="pidtagcallbacktelephonenumber-canonical-property"></a>Kanonische Pidtagcallbacktelephonenumber (-Eigenschaft

  
  
**Gilt für**: Outlook 2013 | Outlook 2016 
  
Enthält eine Telefonnummer, die der Nachrichtenempfänger zum Erreichen des Absenders verwenden kann. 
  
|||
|:-----|:-----|
|Zugeordnete Eigenschaften:  <br/> |PR_CALLBACK_TELEPHONE_NUMBER, PR_CALLBACK_TELEPHONE_NUMBER_A, PR_CALLBACK_TELEPHONE_NUMBER_W  <br/> |
|Kennung:  <br/> |0x3A02  <br/> |
|Datentyp:  <br/> |PT_UNICODE, PT_STRING8  <br/> |
|Bereich:  <br/> |Kontakt  <br/> |
   
## <a name="remarks"></a>Bemerkungen

Diese Eigenschaften sind Beispiele für die Eigenschaften, die die Identifizierung und den Zugriff auf Informationen zu einem Empfänger ermöglichen. Sie werden vom Empfänger und der Organisation des Empfängers definiert.
