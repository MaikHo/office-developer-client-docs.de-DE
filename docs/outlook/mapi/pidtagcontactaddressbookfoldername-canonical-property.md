# <a name="pidtagcontactaddressbookfoldername-canonical-property"></a>Kanonische Pidtagcontactaddressbookfoldername (-Eigenschaft

  
  
**Gilt für**: Outlook 2013 | Outlook 2016 
  
Enthält einen Ordnernamen für Adressbucheinträge.
  
|||
|:-----|:-----|
|Zugeordnete Eigenschaften:  <br/> |PR_CONTAB_FOLDER_NAME, PR_CONTAB_FOLDER_NAME_W  <br/> |
|Kennung:  <br/> |0x6613  <br/> |
|Datentyp:  <br/> |PT_UNICODE, PT_STRING8  <br/> |
|Bereich:  <br/> |Kontakt Adressbuch  <br/> |
   
## <a name="remarks"></a>Bemerkungen

Die folgenden Zeichen können in Ordnernamen nicht verwendet werden:
  
[ ] / \ &amp; ~ ? \* | \<\> " ; : +
