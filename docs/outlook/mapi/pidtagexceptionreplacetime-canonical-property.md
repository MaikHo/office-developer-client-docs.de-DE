# <a name="pidtagexceptionreplacetime-canonical-property"></a>Kanonische Pidtagexceptionreplacetime (-Eigenschaft

  
  
**Gilt für**: Outlook 2013 | Outlook 2016 
  
Gibt das ursprüngliche Datum und die Uhrzeit an, zu der die Instanz im Serienmuster aufgetreten wäre, wenn es sich nicht um eine Ausnahme handelt.
  
|||
|:-----|:-----|
|Zugeordnete Eigenschaften:  <br/> |PR_EXCEPTION_REPLACETIME  <br/> |
|Kennung:  <br/> |0x7FF9  <br/> |
|Datentyp:  <br/> |PT_SYSTIME  <br/> |
|Bereich:  <br/> |Nachrichtenklassen definierte nicht übertragbare Elemente  <br/> |
   
## <a name="remarks"></a>Bemerkungen

Dieser Wert muss in koordinierter weltZeit (UTC) angegeben werden.
