# <a name="pidtaginternetcodepage-canonical-property"></a>PidTagInternetCodepage (kanonische Eigenschaft)

  
  
**Betrifft**: Outlook 
  
Gibt die Codepage für **PR_BODY** ([PidTagBody](pidtagbody-canonical-property.md)) oder **PR_BODY_HTML** ([PidTagBodyHtml](pidtagbodyhtml-canonical-property.md)) Eigenschaften an.
  
|||
|:-----|:-----|
|Zugeordneten Eigenschaften:  <br/> |PR_INTERNET_CPID  <br/> |
|Bezeichner:  <br/> |0x3FDE  <br/> |
|Datentyp:  <br/> |PT_LONG  <br/> |
|Bereich:  <br/> |Verschiedenes  <br/> |
