# <a name="pidlidappointmentreplytime-canonical-property"></a>Kanonische Pidlidappointmentreplytime (-Eigenschaft

  
  
**Gilt für**: Outlook 2013 | Outlook 2016 
  
Gibt das Datum und die Uhrzeit an, zu denen der Teilnehmer auf eine empfangene Besprechungsanfrage oder ein Besprechungs Update geantwortet hat.
  
|||
|:-----|:-----|
|Zugeordnete Eigenschaften:  <br/> |dispidApptReplyTime  <br/> |
|Eigenschaftensatz:  <br/> |PSETID_Appointment  <br/> |
|Long-ID (Deckel):  <br/> |0x00008220  <br/> |
|Datentyp:  <br/> |PT_SYSTIME  <br/> |
|Bereich:  <br/> |Meetings  <br/> |
   
## <a name="remarks"></a>Bemerkungen

Der Wert muss in koordinierter weltZeit (UTC) angegeben werden.
