# <a name="pidlidclassificationkeep-canonical-property"></a>Kanonische Pidlidclassificationkeep (-Eigenschaft

  
  
**Gilt für**: Outlook 2013 | Outlook 2016 
  
Gibt an, ob die Nachrichtenklassifikation mit der Nachricht beibehalten werden soll, wenn die Nachricht weitergeleitet oder geantwortet wird.
  
|||
|:-----|:-----|
|Zugeordnete Eigenschaften:  <br/> |dispidClassKeep  <br/> |
|Eigenschaftensatz:  <br/> |PSETID_Common  <br/> |
|Long-ID (Deckel):  <br/> |0x000085BA  <br/> |
|Datentyp:  <br/> |PT_BOOLEAN  <br/> |
|Bereich:  <br/> |Allgemeine Nachrichtenübermittlung  <br/> |
   
## <a name="remarks"></a>Bemerkungen

Weitere Informationen finden Sie unter [Bereitstellen der Nachrichtenklassifikation für Outlook 2007](https://msdn.microsoft.com/library/5a220424-edd5-4a21-b7fd-8106c23c3b39.aspx).
