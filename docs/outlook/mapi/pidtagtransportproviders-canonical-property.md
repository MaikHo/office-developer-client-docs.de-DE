# <a name="pidtagtransportproviders-canonical-property"></a>PidTagTransportProviders (kanonische Eigenschaft)

  
  
**Betrifft**: Outlook 2013 | Outlook 2016 
  
Enthält eine Liste mit den IDs der Transportanbieter im aktuellen Profil.
  
|||
|:-----|:-----|
|Zugeordnete Eigenschaften:  <br/> |PR_TRANSPORT_PROVIDERS  <br/> |
|Kennung:  <br/> |0x3D02  <br/> |
|Datentyp:  <br/> |PT_BINARY  <br/> |
|Bereich:  <br/> |MAPI-Profil  <br/> |
   
## <a name="remarks"></a>HinwBemerkungeneise

Verwenden Sie diese Eigenschaft nicht. Es ist für die Verwendung von MAPI reserviert.
