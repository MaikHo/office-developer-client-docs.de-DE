# <a name="pidtaggeneration-canonical-property"></a>Kanonische Pidtaggeneration (-Eigenschaft

  
  
**Gilt für**: Outlook 2013 | Outlook 2016 
  
Enthält eine Generationen Abkürzung, die dem vollständigen Namen des Empfängers entspricht. 
  
|||
|:-----|:-----|
|Zugeordnete Eigenschaften:  <br/> |PR_GENERATION, PR_GENERATION_A, PR_GENERATION_W  <br/> |
|Kennung:  <br/> |0x3A05  <br/> |
|Datentyp:  <br/> |PT_STRING8, PT_UNICODE  <br/> |
|Bereich:  <br/> |MAPI-e-Mail-Benutzer  <br/> |
   
## <a name="remarks"></a>Bemerkungen

Diese Eigenschaften bieten Identifizierung und Zugriff auf Informationen zu einem Empfänger. Sie werden vom Empfänger und dessen Organisation definiert. 
  
Zu den allgemeinen Werten gehört Jr., SR. und III.
