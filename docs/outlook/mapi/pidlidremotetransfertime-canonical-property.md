# <a name="pidlidremotetransfertime-canonical-property"></a>PidLidRemoteTransferTime (kanonische Eigenschaft)

  
  
**Betrifft**: Outlook 
  
Gibt die Zeitspanne weiterleiten für ein Remoteelement in Sekunden an.
  
|||
|:-----|:-----|
|Zugeordneten Eigenschaften:  <br/> |dispidRemoteXferTime  <br/> |
|-Eigenschaft festgelegt:  <br/> |PSETID_Remote  <br/> |
|Long-ID (Abdeckung):  <br/> |0x00008F04  <br/> |
|Datentyp:  <br/> |PT_LONG  <br/> |
|Bereich:  <br/> |Remote-Nachricht  <br/> |
