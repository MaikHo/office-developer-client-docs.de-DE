# <a name="attoriginalmessageclass"></a>attOriginalMessageClass

**Betrifft**: Outlook 2013 | Outlook 2016 
  
Eine Nachrichtenklasse wird als Zeichenfolge gespeichert. Die codierte Zeichenfolge enthält in der Regel der MAPI-angegebene Name der Nachrichtenklasse.
