# <a name="pidtagotheraddressstateorprovince-canonical-property"></a>PidTagOtherAddressStateOrProvince (kanonische Eigenschaft)

  
  
**Gilt für**: Outlook 2013 | Outlook 2016 
  
Enthält den Namen des Bundesland/Kanton in der anderen Adresse verwendet.
  
|||
|:-----|:-----|
|Zugeordnete Eigenschaften:  <br/> |PR_OTHER_ADDRESS_STATE_OR_PROVINCE, PR_OTHER_ADDRESS_STATE_OR_PROVINCE_A, PR_OTHER_ADDRESS_STATE_OR_PROVINCE_W  <br/> |
|Kennung:  <br/> |0x3A62  <br/> |
|Datentyp:  <br/> |PT_UNICODE PT_STRING8  <br/> |
|Bereich:  <br/> |Adresse  <br/> |
   
## <a name="remarks"></a>Hinweise

Diese Eigenschaften werden durch den Benutzer oder die Organisation des Benutzers definiert.
