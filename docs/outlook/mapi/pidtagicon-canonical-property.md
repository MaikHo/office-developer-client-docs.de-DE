# <a name="pidtagicon-canonical-property"></a>Kanonische Pidtagicon (-Eigenschaft

  
  
**Gilt für**: Outlook 2013 | Outlook 2016 
  
Enthält eine Bitmap eines vollständigen Symbols für ein Formular. 
  
|||
|:-----|:-----|
|Zugeordnete Eigenschaften:  <br/> |PR_ICON  <br/> |
|Kennung:  <br/> |0x0FFD  <br/> |
|Datentyp:  <br/> |PT_BINARY  <br/> |
|Bereich:  <br/> |Nicht transmitable MAPI  <br/> |
   
## <a name="remarks"></a>Bemerkungen

Diese Eigenschaft enthält ein 32 × 32-Pixel Bild eines Symbols, dasselbe wie der Inhalt von a. ICO-Datei. Diese Eigenschaft wird normalerweise aus der kopiert. ICO-Datei, die in der LargeIcon-Codezeile im entsprechenden Abschnitt [Description] der Formular Konfigurationsdatei angegeben ist.
