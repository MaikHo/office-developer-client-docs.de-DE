# <a name="mapisvcinf-message-service-sections"></a>Nachrichtendienstabschnitte in MapiSvc.inf

  
  
**Betrifft**: Outlook 2013 | Outlook 2016 
  
MAPISVC.inf enthält eine Nachricht Service Abschnitt für die einzelnen Einträge im Abschnitt **[Services]** . Es gibt zwei Arten von Einträgen in den folgenden Abschnitten: eine für das Einrichten von bestimmte Eigenschaften und die andere für die Auflistung von Namen der Abschnitte, die im Zusammenhang mit der Nachrichtendienst konfiguriert wird.
