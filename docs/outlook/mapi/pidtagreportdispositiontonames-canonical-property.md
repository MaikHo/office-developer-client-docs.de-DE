# <a name="pidtagreportdispositiontonames-canonical-property"></a>Kanonische Pidtagreportdispositiontonames (-Eigenschaft

  
  
**Gilt für**: Outlook 2013 | Outlook 2016 
  
Gibt die Namen der Berichtsempfänger für Nachrichten an, die Empfangsbestätigungen anfordern.
  
|||
|:-----|:-----|
|Zugeordnete Eigenschaften:  <br/> |PR_REPORT_DISPOSITION_TO_NAMES, PR_REPORT_DISPOSITION_TO_NAMES_A, PR_REPORT_DISPOSITION_TO_NAMES_W  <br/> |
|Kennung:  <br/> |0x0083  <br/> |
|Datentyp:  <br/> |PT_MV_STRING8, PT_UNICODE  <br/> |
|Bereich:  <br/> |MAPI-Umschlag  <br/> |
