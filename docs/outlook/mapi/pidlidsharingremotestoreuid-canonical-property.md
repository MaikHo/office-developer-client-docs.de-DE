# <a name="pidlidsharingremotestoreuid-canonical-property"></a>Kanonische PidLidSharingRemoteStoreUid-Eigenschaft

  
  
**Betrifft**: Outlook 
  
Gibt die hexadezimale Zeichenfolgendarstellung des Werts der Eigenschaft **PR_STORE_ENTRYID** ([PidTagStoreEntryId](pidtagstoreentryid-canonical-property.md)) für den freigegebenen Ordner an. Dies ist eine Eigenschaft eines eine Freigabenachricht.
  
|||
|:-----|:-----|
|Zugeordneten Eigenschaften:  <br/> |dispidSharingRemoteStoreUid  <br/> |
|-Eigenschaft festgelegt:  <br/> |PSETID_Sharing  <br/> |
|Long-ID (Abdeckung):  <br/> |0x00008A48  <br/> |
|Datentyp:  <br/> |PT_UNICODE  <br/> |
|Bereich:  <br/> |Freigabe  <br/> |
