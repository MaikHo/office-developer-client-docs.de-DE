# <a name="pidtagcontactaddresstypes-canonical-property"></a>Kanonische PidTagContactAddressTypes-Eigenschaft

  
  
**Betrifft**: Outlook 
  
Enthält eine Liste von Adresstypen für einen Kontakt.
  
|||
|:-----|:-----|
|Zugeordneten Eigenschaften:  <br/> |PR_CONTACT_ADDRTYPES, PR_CONTACT_ADDRTYPES_A, PR_CONTACT_ADDRTYPES_W  <br/> |
|Bezeichner:  <br/> |0x3A54  <br/> |
|Datentyp:  <br/> |PT_MV_UNICODE PT_MV_STRING8  <br/> |
|Bereich:  <br/> |Kontakt  <br/> |
