# <a name="pidtagformhidden-canonical-property"></a>Kanonische Pidtagformhidden (-Eigenschaft

  
  
**Gilt für**: Outlook 2013 | Outlook 2016 
  
Enthält TRUE, wenn ein Formular unterdrückt werden soll, indem Sie Menüs und Dialogfelder verfassen. 
  
|||
|:-----|:-----|
|Zugeordnete Eigenschaften:  <br/> |PR_FORM_HIDDEN  <br/> |
|Kennung:  <br/> |0x3307  <br/> |
|Datentyp:  <br/> |PT_BOOLEAN  <br/> |
|Bereich:  <br/> |MAPI allgemein  <br/> |
   
## <a name="remarks"></a>Bemerkungen

Formularbezogene Eigenschaften sind schreibgeschützt.
