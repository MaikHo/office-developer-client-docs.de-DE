# <a name="pidtagformhidden-canonical-property"></a>PidTagFormHidden (kanonische Eigenschaft)

  
  
**Betrifft**: Outlook 2013 | Outlook 2016 
  
Enthält True, wenn ein Formular aus der Anzeige unterdrückt werden durch Verfassen, Menüs und Dialogfelder. 
  
|||
|:-----|:-----|
|Zugeordnete Eigenschaften:  <br/> |PR_FORM_HIDDEN  <br/> |
|Kennung:  <br/> |0x3307  <br/> |
|Datentyp:  <br/> |PT_BOOLEAN  <br/> |
|Bereich:  <br/> |Allgemeine MAPI  <br/> |
   
## <a name="remarks"></a>HinwBemerkungeneise

Formular-bezogene Eigenschaften sind schreibgeschützt.
