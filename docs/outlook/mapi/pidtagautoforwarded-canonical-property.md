# <a name="pidtagautoforwarded-canonical-property"></a>Kanonische Pidtagautoforwarded (-Eigenschaft

  
  
**Gilt für**: Outlook 2013 | Outlook 2016 
  
Enthält TRUE, wenn der Client ein Kopfzeilenfeld X-MS-Exchange-Organization-autoForward anfordert.
  
|||
|:-----|:-----|
|Zugeordnete Eigenschaften:  <br/> |PR_AUTO_FORWARDED  <br/> |
|Kennung:  <br/> |0x0005  <br/> |
|Datentyp:  <br/> |PT_BOOLEAN  <br/> |
|Bereich:  <br/> |Allgemeine Berichterstellung  <br/> |
   
## <a name="remarks"></a>Bemerkungen

Wenn diese Eigenschaft auf FALSE festgelegt oder nicht verwendet wird, wird kein X-MS-Exchange-Organization-Auto forwarded-Kopfzeilenfeld erstellt.
