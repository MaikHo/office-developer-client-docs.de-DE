# <a name="pidtagsearchfolderid-canonical-property"></a>Kanonische Pidtagsearchfolderid (-Eigenschaft

  
  
**Gilt für**: Outlook 2013 | Outlook 2016 
  
Enthält eine GUID (Globally Unique Identifier), die den Suchordner identifiziert.
  
|||
|:-----|:-----|
|Zugeordnete Eigenschaften:  <br/> |PR_WB_SF_ID  <br/> |
|Kennung:  <br/> |0x6842  <br/> |
|Datentyp:  <br/> |PT_BINARY  <br/> |
|Bereich:  <br/> |Suche  <br/> |
