# <a name="pidtagcontentreturnrequested-canonical-property"></a>Kanonische PidTagContentReturnRequested-Eigenschaft

  
  
**Betrifft**: Outlook 
  
Enthält True, wenn eine Nachricht mit einem Unzustellbarkeitsbericht zurückgegeben werden sollen. 
  
|||
|:-----|:-----|
|Zugeordneten Eigenschaften:  <br/> |PR_CONTENT_RETURN_REQUESTED  <br/> |
|Bezeichner:  <br/> |0x000a  <br/> |
|Datentyp:  <br/> |PT_BOOLEAN  <br/> |
|Bereich:  <br/> |Report (Bericht)  <br/> |
   
## <a name="remarks"></a>Bemerkungen

Wenn diese Eigenschaft nicht festgelegt ist, wird Sie MAPI wie mit dem Wert TRUE behandelt.
