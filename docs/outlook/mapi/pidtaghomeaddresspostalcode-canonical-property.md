# <a name="pidtaghomeaddresspostalcode-canonical-property"></a>Kanonische PidTagHomeAddressPostalCode-Eigenschaft

  
  
**Betrifft**: Outlook 
  
Enthält die Postleitzahl für Privatanschrift des Benutzers an.
  
|||
|:-----|:-----|
|Zugeordneten Eigenschaften:  <br/> |PR_HOME_ADDRESS_POSTAL_CODE, PR_HOME_ADDRESS_POSTAL_CODE_A, PR_HOME_ADDRESS_POSTAL_CODE_W  <br/> |
|Bezeichner:  <br/> |0x3A5B  <br/> |
|Datentyp:  <br/> |PT_STRING8, PT_UNICODE  <br/> |
|Bereich:  <br/> |MAPI-e-Mail-Benutzer  <br/> |
   
## <a name="remarks"></a>Hinweise

Diese Eigenschaften werden durch den Benutzer oder die Organisation des Benutzers definiert.
