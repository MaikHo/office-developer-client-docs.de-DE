# <a name="pidtagnondeliveryreportfromentryid-canonical-property"></a>Kanonische Pidtagnondeliveryreportfromentryid (-Eigenschaft

  
  
**Gilt für**: Outlook 2013 | Outlook 2016 
  
Wenn für einen Unzustellbarkeitsbericht festgelegt, wird die Eintrags-ID des fehlgeschlagenen senden als Benutzer.
  
|||
|:-----|:-----|
|Zugeordnete Eigenschaften:  <br/> |PR_NDR_FROM_ENTRYID  <br/> |
|Kennung:  <br/> |0x6904  <br/> |
|Datentyp:  <br/> |PT_BINARY  <br/> |
|Bereich:  <br/> |Allgemeine Nachrichtenübermittlung  <br/> |
