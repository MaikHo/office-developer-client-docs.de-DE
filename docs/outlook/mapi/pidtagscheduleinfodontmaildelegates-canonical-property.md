# <a name="pidtagscheduleinfodontmaildelegates-canonical-property"></a>Kanonische Pidtagscheduleinfodontmaildelegates (-Eigenschaft

  
  
**Gilt für**: Outlook 2013 | Outlook 2016 
  
Enthält TRUE, wenn der Stellvertreter keine Updates erhalten möchte.
  
|||
|:-----|:-----|
|Zugeordnete Eigenschaften:  <br/> |PR_SCHDINFO_DONT_MAIL_DELEGATES  <br/> |
|Kennung:  <br/> |0x6843  <br/> |
|Datentyp:  <br/> |PT_BOOLEAN  <br/> |
|Bereich:  <br/> |Frei/Gebucht  <br/> |
   
## <a name="remarks"></a>Bemerkungen

Diese Eigenschaft muss im Delegate Information-Objekt festgelegt werden.
