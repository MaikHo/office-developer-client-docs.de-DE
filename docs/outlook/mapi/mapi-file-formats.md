# <a name="mapi-file-formats"></a>MAPI-Dateiformate

  
  
**Betrifft**: Outlook 
  
MAPI veröffentlicht die Formate der verschiedenen Konfigurationsdateien, mit denen Clients und Dienstanbieter Message-Dienste und deren-Dienstanbieter und Formulare beschrieben werden.
  
In der folgenden Tabelle werden diese Dateien nach Name oder Typ beschrieben:
  
|**Konfigurationsdatei**|**Beschreibung**|
|:-----|:-----|
|MapiSvc.inf  <br/> |Enthält Konfigurationsinformationen für die MAPI-Subsystem, Message-Dienste und -Dienstanbieter.  <br/> |
|Dateien mit der Erweiterung cfg  <br/> |Konfigurationsinformationen für ein Formular enthält.  <br/> |
   
Ausführliche Informationen zum Dateiformat MapiSvc.inf finden Sie unter [File Format der MapiSvc.inf](file-format-of-mapisvc-inf.md). 
  
Ausführliche Informationen zum Dateiformat einer Konfigurationsdatei Formular finden Sie unter [File Format des Formulars Konfigurationsdateien](file-format-of-form-configuration-files.md).
