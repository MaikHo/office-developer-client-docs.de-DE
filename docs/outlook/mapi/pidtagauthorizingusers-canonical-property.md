# <a name="pidtagauthorizingusers-canonical-property"></a>PidTagAuthorizingUsers (kanonische Eigenschaft)

  
  
**Betrifft**: Outlook 2013 | Outlook 2016 
  
Enthält eine Liste von Eintragsbezeichner für Benutzer, die autorisiert ist das Senden einer Nachricht an.
  
|||
|:-----|:-----|
|Zugeordnete Eigenschaften:  <br/> |PR_AUTHORIZING_USERS  <br/> |
|Kennung:  <br/> |0 x 0003  <br/> |
|Datentyp:  <br/> |PT_BINARY  <br/> |
|Bereich:  <br/> |Exchange  <br/> |
   
## <a name="remarks"></a>HinwBemerkungeneise

Die Nachrichtenspeichers wird diese Eigenschaft nicht an.
