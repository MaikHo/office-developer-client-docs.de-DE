# <a name="pidtagauthorizingusers-canonical-property"></a>Kanonische Pidtagauthorizingusers (-Eigenschaft

  
  
**Gilt für**: Outlook 2013 | Outlook 2016 
  
Enthält eine Liste von Eintrags Bezeichnern für Benutzer, die das Senden einer Nachricht autorisiert haben.
  
|||
|:-----|:-----|
|Zugeordnete Eigenschaften:  <br/> |PR_AUTHORIZING_USERS  <br/> |
|Kennung:  <br/> |0x0003  <br/> |
|Datentyp:  <br/> |PT_BINARY  <br/> |
|Bereich:  <br/> |Exchange  <br/> |
   
## <a name="remarks"></a>Bemerkungen

Diese Eigenschaft wird vom Nachrichtenspeicher nicht verwaltet.
