# <a name="pidtagtnefunprocessedprops-canonical-property"></a>Kanonische Pidtagtnefunprocessedprops (-Eigenschaft

  
  
**Gilt für**: Outlook 2013 | Outlook 2016 
  
Serialisiert Eigenschaften beim Filtern von TNEF (Transport Neutral Encapsulation Format).
  
|||
|:-----|:-----|
|Zugeordnete Eigenschaften:  <br/> |PR_TNEF_UNPROCESSED_PROPS  <br/> |
|Kennung:  <br/> |0x0E9C  <br/> |
|Datentyp:  <br/> |PT_BINARY  <br/> |
|Bereich:  <br/> |Nicht transmitable MAPI  <br/> |
   
## <a name="remarks"></a>Bemerkungen

Wird von Microsoft Outlook und Outlook Web Access (OWA) zum Speichern der ursprünglichen TNEF in den Fällen verwendet, in denen TNEF benannte Eigenschaften enthält, die im Speicher nicht erstellt werden können.
