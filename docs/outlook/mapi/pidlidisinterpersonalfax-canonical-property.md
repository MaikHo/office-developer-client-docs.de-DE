# <a name="pidlidisinterpersonalfax-canonical-property"></a>PidLidIsInterpersonalFax (kanonische Eigenschaft)

  
  
**Betrifft**: Outlook 2013 | Outlook 2016 
  
Gibt an, ob das Element eines Faxelements Internet ist.
  
|||
|:-----|:-----|
|Zugeordnete Eigenschaften:  <br/> |dispidIsIPFax  <br/> |
|-Eigenschaft festgelegt:  <br/> |PSETID_Common  <br/> |
|Long-ID (Abdeckung):  <br/> |0x0000859B  <br/> |
|Datentyp:  <br/> |PT_BOOLEAN  <br/> |
|Bereich:  <br/> |Allgemeine messaging  <br/> |
