# <a name="pidlidinternetaccountname-canonical-property"></a>Kanonische Pidlidinternetaccountname (-Eigenschaft

  
  
**Gilt für**: Outlook 2013 | Outlook 2016 
  
Gibt den Benutzersicht baren e-Mail-Kontonamen an, über den die e-Mail-Nachricht gesendet wird.
  
|||
|:-----|:-----|
|Zugeordnete Eigenschaften:  <br/> |dispidInetAcctName  <br/> |
|Eigenschaftensatz:  <br/> |PSETID_Common  <br/> |
|Long-ID (Deckel):  <br/> |0x00008580  <br/> |
|Datentyp:  <br/> |PT_UNICODE  <br/> |
|Bereich:  <br/> |Allgemeine Nachrichtenübermittlung  <br/> |
   
## <a name="remarks"></a>Bemerkungen

Das Format dieser Zeichenfolge ist implementierungsabhängig. Diese Eigenschaft kann vom Client verwendet werden, um zu bestimmen, auf welchem Server die e-Mail umgeleitet werden soll, ist jedoch optional, und der Wert hat keine Bedeutung für den Server.
