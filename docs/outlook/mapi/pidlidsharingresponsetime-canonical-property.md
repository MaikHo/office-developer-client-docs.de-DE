# <a name="pidlidsharingresponsetime-canonical-property"></a>PidLidSharingResponseTime (kanonische Eigenschaft)

  
  
**Betrifft**: Outlook 2013 | Outlook 2016 
  
Gibt die Zeit, zu der der Empfänger die Freigabeanfrage eine Freigabeantwort gesendet. Dies ist eine Eigenschaft eines eine Freigabenachricht.
  
|||
|:-----|:-----|
|Zugeordnete Eigenschaften:  <br/> |dispidSharingResponseTime  <br/> |
|-Eigenschaft festgelegt:  <br/> |PSETID_Sharing  <br/> |
|Long-ID (Abdeckung):  <br/> |0x00008A28  <br/> |
|Datentyp:  <br/> |PT_SYSTIME  <br/> |
|Bereich:  <br/> |Freigabe  <br/> |
