# <a name="pidnameaudionotes-canonical-property"></a>Kanonische Pidnameaudionotes (-Eigenschaft

  
  
**Gilt für**: Outlook 2013 | Outlook 2016 
  
Gibt die Textnotizen an, die an eine Sprachnachricht angefügt sind.
  
|||
|:-----|:-----|
|Angezeigte Namen:  <br/> |UMAudioNotes  <br/> |
|Eigenschaftensatz:  <br/> |PSETID_UnifiedMessaging  <br/> |
|Eigenschaftsname:  <br/> |UMAudioNotes  <br/> |
|Datentyp:  <br/> |PT_UNICODE  <br/> |
|Bereich:  <br/> |Unified Messaging  <br/> |
   
## <a name="remarks"></a>Bemerkungen

Um einem Endbenutzer das Lesen und Bearbeiten von Audionotizen direkt in einer Sprachnachricht zu ermöglichen, stellt ein Client ein Bearbeitungsfeld bereit, in dem der Benutzer einen Satz von Notizen eingeben kann, die dieser Eigenschaft des Sprachnachrichten Objekts hinzugefügt werden.
