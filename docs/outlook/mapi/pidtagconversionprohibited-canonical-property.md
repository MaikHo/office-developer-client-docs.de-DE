# <a name="pidtagconversionprohibited-canonical-property"></a>Kanonische Pidtagconversionprohibited (-Eigenschaft

  
  
**Gilt für**: Outlook 2013 | Outlook 2016 
  
Enthält TRUE, wenn Nachrichten Konvertierungen standardmäßig für den zugeordneten Messagingbenutzer unzulässig sind.
  
|||
|:-----|:-----|
|Zugeordnete Eigenschaften:  <br/> |PR_CONVERSION_PROHIBITED  <br/> |
|Kennung:  <br/> |0x3A03  <br/> |
|Datentyp:  <br/> |PT_BOOLEAN  <br/> |
|Bereich:  <br/> |Adresse  <br/> |
