# <a name="pidtaghobbies-canonical-property"></a>Kanonische Pidtaghobbies (-Eigenschaft

  
  
**Gilt für**: Outlook 2013 | Outlook 2016 
  
Enthält die Namen der Hobbies des Messaging Benutzers.
  
|||
|:-----|:-----|
|Zugeordnete Eigenschaften:  <br/> |PR_HOBBIES, PR_HOBBIES_A, PR_HOBBIES_W  <br/> |
|Kennung:  <br/> |0x3A43  <br/> |
|Datentyp:  <br/> |PT_STRING8, PT_UNICODE  <br/> |
|Bereich:  <br/> |MAPI-e-Mail-Benutzer  <br/> |
   
## <a name="remarks"></a>Bemerkungen

Diese Eigenschaften bieten Identifizierung und Zugriff auf Informationen zu einem Messagingbenutzer, und der Inhalt wird von der Organisation des Benutzers definiert.
