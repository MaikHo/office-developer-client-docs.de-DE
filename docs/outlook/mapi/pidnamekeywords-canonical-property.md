# <a name="pidnamekeywords-canonical-property"></a>Kanonische Pidnamekeywords (-Eigenschaft

  
  
**Gilt für**: Outlook 2013 | Outlook 2016 
  
Enthält Schlüsselwörter oder Kategorien für das Message-Objekt.
  
|||
|:-----|:-----|
|Angezeigte Namen:  <br/> |Keine  <br/> |
|Eigenschaftensatz:  <br/> |PS_PUBLIC_STRINGS  <br/> |
|Eigenschaftsname:  <br/> |Schlüsselwörter  <br/> |
|Datentyp:  <br/> |PT_MV_UNICODE  <br/> |
|Bereich:  <br/> |Allgemeine Nachrichtenübermittlung  <br/> |
   
## <a name="remarks"></a>Bemerkungen

Ein mehrwertiger String-Wert, der die Kategorien für ein Nachrichtenobjekt angibt, die Länge der einzelnen Zeichenfolgen innerhalb dieser Eigenschaft mit mehrwertiger Zeichenfolge muss kleiner als 256 sein.
