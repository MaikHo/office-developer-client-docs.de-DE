# <a name="pidlidnoteheight-canonical-property"></a>Kanonische PidLidNoteHeight-Eigenschaft

  
  
**Betrifft**: Outlook 
  
Gibt die Höhe des Fensters sichtbar Nachricht in Pixeln an. 
  
|||
|:-----|:-----|
|Zugeordneten Eigenschaften:  <br/> |dispidNoteHeight  <br/> |
|-Eigenschaft festgelegt:  <br/> |PSETID_Note  <br/> |
|Long-ID (Abdeckung):  <br/> |0x00008B03  <br/> |
|Datentyp:  <br/> |PT_LONG  <br/> |
|Bereich:  <br/> |Kurznotizen  <br/> |
   
## <a name="remarks"></a>Hinweise

Dieser Wert muss größer als NULL sein.
