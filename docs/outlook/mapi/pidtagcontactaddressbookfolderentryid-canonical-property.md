# <a name="pidtagcontactaddressbookfolderentryid-canonical-property"></a>Kanonische Pidtagcontactaddressbookfolderentryid (-Eigenschaft

  
  
**Gilt für**: Outlook 2013 | Outlook 2016 
  
Enthält einen Bezeichner für den Ordner Kontakte im Nachrichtenspeicher, in dem e-Mail-Benutzer aufbewahrt werden. 
  
|||
|:-----|:-----|
|Zugeordnete Eigenschaften:  <br/> |PR_CONTAB_FOLDER_ENTRYID  <br/> |
|Kennung:  <br/> |0x6610  <br/> |
|Datentyp:  <br/> |PT_BINARY  <br/> |
|Bereich:  <br/> |Kontakt Adressbuch  <br/> |
   
## <a name="remarks"></a>Bemerkungen

Hierbei handelt es sich um eine Eigenschaft in einem Adressbuchcontainer des Kontakts und eine Spalte in der Tabelle mit Adressbuch Containern des Kontakts.
