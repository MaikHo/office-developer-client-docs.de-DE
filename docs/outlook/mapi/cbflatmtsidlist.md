**Betrifft**: Outlook 2013 | Outlook 2016

|||
|:-----|:-----|
|Headerdatei  <br/> |Mapidefs.h  <br/> |
|Verwandte Struktur:  <br/> |**FLATMTSIDLIST** <br/> |
   
```cpp
CbFLATMTSIDLIST (_lplist)
```

## <a name="parameters"></a>Parameter
