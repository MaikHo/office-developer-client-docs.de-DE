# <a name="pidlidagingdontageme-canonical-property"></a>PidLidAgingDontAgeMe (kanonische Eigenschaft)

  
  
**Betrifft**: Outlook 
  
Gibt den Wert TRUE an, wenn das Element nicht automatisch archiviert werden.
  
|||
|:-----|:-----|
|Zugeordneten Eigenschaften:  <br/> |dispidAgingDontAgeMe  <br/> |
|-Eigenschaft festgelegt:  <br/> |PSETID_Common  <br/> |
|Long-ID (Abdeckung):  <br/> |0x0000850E  <br/> |
|Datentyp:  <br/> |PT_BOOLEAN  <br/> |
|Bereich:  <br/> |Common  <br/> |
