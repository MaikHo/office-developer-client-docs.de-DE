# <a name="pidtaginternetarticlenumber-canonical-property"></a>Kanonische Pidtaginternetarticlenumber (-Eigenschaft

  
  
**Gilt für**: Outlook 2013 | Outlook 2016 
  
Eine Zahl, die einem Element in einem Nachrichtenspeicher zugeordnet ist.
  
|||
|:-----|:-----|
|Zugeordnete Eigenschaften:  <br/> |PR_INTERNET_ARTICLE_NUMBER  <br/> |
|Kennung:  <br/> |0x0E23  <br/> |
|Datentyp:  <br/> |PT_LONG  <br/> |
|Bereich:  <br/> |MIME  <br/> |
   
## <a name="remarks"></a>Bemerkungen

In IMAPi-Nachrichten speichern entspricht diese Eigenschaft der IMAPi-UID.
