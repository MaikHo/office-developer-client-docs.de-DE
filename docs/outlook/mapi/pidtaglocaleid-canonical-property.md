# <a name="pidtaglocaleid-canonical-property"></a>Kanonische Pidtaglocaleid (-Eigenschaft

  
  
**Gilt für**: Outlook 2013 | Outlook 2016 
  
Legt das Gebietsschema für die Sprachumgebung für die Übersetzung von vom System generierten Nachrichten (beispielsweise zugestellt) fest.
  
|||
|:-----|:-----|
|Zugeordnete Eigenschaften:  <br/> |PR_LOCALE_ID  <br/> |
|Kennung:  <br/> |0x66A1  <br/> |
|Datentyp:  <br/> |PT_LONG  <br/> |
|Bereich:  <br/> |Sonstiges  <br/> |
   
## <a name="remarks"></a>Bemerkungen

Diese Eigenschaft ist für eine private Postfachanmeldung verfügbar.
