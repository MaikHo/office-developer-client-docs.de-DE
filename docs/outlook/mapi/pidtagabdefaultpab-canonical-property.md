# <a name="pidtagabdefaultpab-canonical-property"></a>PidTagAbDefaultPab (kanonische Eigenschaft)

  
  
**Betrifft**: Outlook 2013 | Outlook 2016 
  
Enthält die Eintrags-ID der Adressbuchcontainer als das persönliche Adressbuch (PAB) verwenden. 
  
|||
|:-----|:-----|
|Zugeordnete Eigenschaften:  <br/> |PR_AB_DEFAULT_PAB  <br/> |
|Kennung:  <br/> |0x3D07  <br/> |
|Datentyp:  <br/> |PT_BINARY  <br/> |
|Bereich:  <br/> |Adressbuch  <br/> |
   
## <a name="remarks"></a>HinwBemerkungeneise

Verwenden Sie diese Eigenschaft nicht. Es ist für die Verwendung von MAPI reserviert.
