<span data-ttu-id="9cfa3-105">Adressbuchanbieter sind für die Unterstützung der Namensauflösung verantwortlich – der Prozess der Zuordnung eines Eintrags Bezeichners zu einem Anzeigenamen.</span><span class="sxs-lookup"><span data-stu-id="9cfa3-105">Address book providers are responsible for supporting name resolution — the process of associating an entry identifier with a display name.</span></span> <span data-ttu-id="9cfa3-106">Clients initiieren die Namensauflösung, wenn Sie [IAddrBook::](iaddrbook-resolvename.md) ResolveName aufrufen, um sicherzustellen, dass jedes Mitglied der Empfängerliste einer ausgehenden Nachricht einer gültigen Adresse entspricht.</span><span class="sxs-lookup"><span data-stu-id="9cfa3-106">Clients initiate name resolution when they call [IAddrBook::ResolveName](iaddrbook-resolvename.md) to ensure that each member of an outgoing message's recipient list corresponds to a valid address.</span></span>
