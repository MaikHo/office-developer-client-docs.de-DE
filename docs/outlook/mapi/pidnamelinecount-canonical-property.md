# <a name="pidnamelinecount-canonical-property"></a>Kanonische PidNameLineCount-Eigenschaft

  
  
**Betrifft**: Outlook 
  
Gibt die Anzahl der Zeilen in der Datei, die an das Dokument angefügt ist.
  
|||
|:-----|:-----|
|Anzeigenamen:  <br/> |Keine  <br/> |
|-Eigenschaft festgelegt:  <br/> |PS_PUBLIC_STRINGS  <br/> |
|Name der Eigenschaft:  <br/> |LineCount  <br/> |
|Datentyp:  <br/> |PT_LONG  <br/> |
|Bereich:  <br/> |Common  <br/> |
