# <a name="pidtaginternetmessageid-canonical-property"></a>Kanonische Pidtaginternetmessageid (-Eigenschaft

  
  
**Gilt für**: Outlook 2013 | Outlook 2016 
  
Entspricht dem Feld Nachrichten-ID gemäß [RFC2822].
  
|||
|:-----|:-----|
|Zugeordnete Eigenschaften:  <br/> |PR_INTERNET_MESSAGE_ID, PR_INTERNET_MESSAGE_ID_A, PR_INTERNET_MESSAGE_ID_W  <br/> |
|Kennung:  <br/> |0x1035  <br/> |
|Datentyp:  <br/> |PT_STRING8, PT_UNICODE  <br/> |
|Bereich:  <br/> |MIME  <br/> |
   
## <a name="remarks"></a>Bemerkungen

Diese Eigenschaften sollten in allen e-Mail-Nachrichten vorhanden sein.
