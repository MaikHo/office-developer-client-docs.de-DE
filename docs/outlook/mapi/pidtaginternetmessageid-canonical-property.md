# <a name="pidtaginternetmessageid-canonical-property"></a>PidTagInternetMessageId (kanonische Eigenschaft)

  
  
**Betrifft**: Outlook 
  
Im Feld Meldung ID gemäß [RFC2822] entspricht.
  
|||
|:-----|:-----|
|Zugeordneten Eigenschaften:  <br/> |PR_INTERNET_MESSAGE_ID, PR_INTERNET_MESSAGE_ID_A, PR_INTERNET_MESSAGE_ID_W  <br/> |
|Bezeichner:  <br/> |0x1035  <br/> |
|Datentyp:  <br/> |PT_STRING8, PT_UNICODE  <br/> |
|Bereich:  <br/> |MIME  <br/> |
   
## <a name="remarks"></a>Bemerkungen

Diese Eigenschaften sollten für alle e-Mail-Nachrichten vorhanden sein.
