# <a name="pidtagpostaladdress-canonical-property"></a>Kanonische Pidtagpostaladdress (-Eigenschaft

  
  
**Gilt für**: Outlook 2013 | Outlook 2016 
  
Enthält die Postanschrift des Empfängers.
  
|||
|:-----|:-----|
|Zugeordnete Eigenschaften:  <br/> |PR_POSTAL_ADDRESS, PR_POSTAL_ADDRESS_A, PR_POSTAL_ADDRESS_W  <br/> |
|Kennung:  <br/> |0x3A15  <br/> |
|Datentyp:  <br/> |PT_STRING8, PT_UNICODE  <br/> |
|Bereich:  <br/> |MAPI-e-Mail-Benutzer  <br/> |
   
## <a name="remarks"></a>Bemerkungen

Diese Eigenschaften bieten Identifizierungs-und Zugriffsinformationen für einen Empfänger. Sie werden vom Empfänger und der Organisation des Empfängers definiert.
