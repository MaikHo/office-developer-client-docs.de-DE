# <a name="pidtagspamtrustedrecipients-canonical-property"></a>Kanonische PidTagSpamTrustedRecipients-Eigenschaft
 
**Betrifft**: Outlook 
  
Enthält eine durch Semikolons getrennte Liste der e-Mail-Adressen und Domänen, die die vertrauenswürdigen Empfänger darstellen.
  
|||
|:-----|:-----|
|Zugeordneten Eigenschaften:  <br/> |PR_SPAM_TRUSTED_RECIPIENTS_W  <br/> |
|Long-ID (Abdeckung):  <br/> |0x0419  <br/> |
|Datentyp:  <br/> |PT_UNICODE  <br/> |
|Bereich:  <br/> |Spam  <br/> |
