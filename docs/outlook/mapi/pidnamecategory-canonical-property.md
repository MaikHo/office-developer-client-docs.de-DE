# <a name="pidnamecategory-canonical-property"></a>Kanonische Pidnamecategory (-Eigenschaft

  
  
**Gilt für**: Outlook 2013 | Outlook 2016 
  
Gibt die Kategorie der an das Dokument angefügten Datei an.
  
|||
|:-----|:-----|
|Angezeigte Namen:  <br/> |Keine  <br/> |
|Eigenschaftensatz:  <br/> |PS_PUBLIC_STRINGS  <br/> |
|Eigenschaftsname:  <br/> |Kategorie  <br/> |
|Datentyp:  <br/> |PT_UNICODE  <br/> |
|Bereich:  <br/> |Standard  <br/> |
