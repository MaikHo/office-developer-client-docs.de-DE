# <a name="pidnamecategory-canonical-property"></a>PidNameCategory (kanonische Eigenschaft)

  
  
**Betrifft**: Outlook 
  
Gibt die Kategorie der Datei mit dem Dokument verbundenen.
  
|||
|:-----|:-----|
|Anzeigenamen:  <br/> |Keine  <br/> |
|-Eigenschaft festgelegt:  <br/> |PS_PUBLIC_STRINGS  <br/> |
|Name der Eigenschaft:  <br/> |Kategorie  <br/> |
|Datentyp:  <br/> |PT_UNICODE  <br/> |
|Bereich:  <br/> |Common  <br/> |
