# <a name="pidlidworkaddresscountry-canonical-property"></a>Kanonische Pidlidworkaddresscountry (-Eigenschaft

  
  
**Gilt für**: Outlook 2013 | Outlook 2016 
  
Gibt den Land-oder Regions Teil der Arbeitsadresse des Kontakts an.
  
|||
|:-----|:-----|
|Zugeordnete Eigenschaften:  <br/> |dispidWorkAddressCountry  <br/> |
|Eigenschaftensatz:  <br/> |PSETID_Address  <br/> |
|Long-ID (Deckel):  <br/> |0x00008049  <br/> |
|Datentyp:  <br/> |PT_UNICODE  <br/> |
|Bereich:  <br/> |Kontakt  <br/> |
