# <a name="pidtagtypeofx400user-canonical-property"></a>Kanonische PidTagTypeOfX400User-Eigenschaft

  
  
**Betrifft**: Outlook 
  
Diese Eigenschaft enthält den Typ der Empfänger einer Nachricht für die Verwendung in einem Bericht.
  
|||
|:-----|:-----|
|Zugeordneten Eigenschaften:  <br/> |PR_TYPE_OF_MTS_USER  <br/> |
|Bezeichner:  <br/> |0x0C1C  <br/> |
|Datentyp:  <br/> |PT_LONG  <br/> |
|Bereich:  <br/> |MAPI-Empfänger  <br/> |
