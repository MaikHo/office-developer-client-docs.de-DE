# <a name="pidlidprivate-canonical-property"></a>PidLidPrivate (kanonische Eigenschaft)

  
  
**Betrifft**: Outlook 2013 | Outlook 2016 
  
Gibt an, ob der Endbenutzer dieses Objekt "Message möchte" andere Benutzer mit Zugriff auf das Objekt "Message" ausgeblendet.
  
|||
|:-----|:-----|
|Zugeordnete Eigenschaften:  <br/> |dispidPrivate  <br/> |
|-Eigenschaft festgelegt:  <br/> |PSETID_Common  <br/> |
|Long-ID (Abdeckung):  <br/> |0x00008506  <br/> |
|Datentyp:  <br/> |PT_BOOLEAN  <br/> |
|Bereich:  <br/> |Allgemeine messaging  <br/> |
