# <a name="pidtagttytddphonenumber-canonical-property"></a>Kanonische Pidtagttytddphonenumber (-Eigenschaft

  
  
**Gilt für**: Outlook 2013 | Outlook 2016 
  
Enthält die Telefonnummer für das Texttelefon des Kontakts (TTY) oder das Telekommunikationsgerät für Gehörlose (TDD).
  
|||
|:-----|:-----|
|Zugeordnete Eigenschaften:  <br/> |PR_TTYTDD_PHONE_NUMBER, PR_TTYTDD_PHONE_NUMBER_A, PR_TTYTDD_PHONE_NUMBER_W  <br/> |
|Kennung:  <br/> |0x3A4B  <br/> |
|Datentyp:  <br/> |PT_STRING8, PT_UNICODE  <br/> |
|Bereich:  <br/> |Kontakt  <br/> |
