# <a name="pidtagttytddphonenumber-canonical-property"></a>Kanonische PidTagTtyTddPhoneNumber-Eigenschaft

  
  
**Betrifft**: Outlook 
  
Enthält die Telefonnummer für den Kontakt Texttelefon (TTY) oder Telekommunikation Gerät für die Gehörlose (TDD).
  
|||
|:-----|:-----|
|Zugeordneten Eigenschaften:  <br/> |PR_TTYTDD_PHONE_NUMBER, PR_TTYTDD_PHONE_NUMBER_A, PR_TTYTDD_PHONE_NUMBER_W  <br/> |
|Bezeichner:  <br/> |0x3A4B  <br/> |
|Datentyp:  <br/> |PT_STRING8, PT_UNICODE  <br/> |
|Bereich:  <br/> |Kontakt  <br/> |
