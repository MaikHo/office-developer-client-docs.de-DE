# <a name="cbmapiformproparray"></a>CbMAPIFormPropArray

  
  
**Betrifft**: Outlook 
  
Berechnet die Anzahl von Bytes in eine vorhandene [SMAPIFormPropArray](smapiformproparray.md) -Struktur, die eine angegebene Anzahl von [SMAPIFormProp](smapiformprop.md) Strukturen enthält. 
  
|||
|:-----|:-----|
|Headerdatei  <br/> |Mapiform.h  <br/> |
|Verwandte Struktur:  <br/> |**SMAPIFormPropArray** <br/> |
   
```cpp
CbMAPIFormPropArray (_c)
```

## <a name="parameters"></a>Parameter

 __c_
  
> Anzahl der **SMAPIFormProp** Strukturen in der betreffenden **SMAPIFormPropArray** Struktur.
