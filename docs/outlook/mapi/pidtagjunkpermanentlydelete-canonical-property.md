# <a name="pidtagjunkpermanentlydelete-canonical-property"></a>Kanonische Pidtagjunkpermanentlydelete (-Eigenschaft

  
  
**Gilt für**: Outlook 2013 | Outlook 2016 
  
Gibt bei Festlegung auf 1 an, dass Nachrichten, die als Spam identifiziert wurden, möglicherweise dauerhaft gelöscht werden.
  
|||
|:-----|:-----|
|Zugeordnete Eigenschaften:  <br/> |PR_JUNK_PERMANENTLY_DELETE  <br/> |
|Kennung:  <br/> |0x6102  <br/> |
|Datentyp:  <br/> |PT_LONG  <br/> |
|Bereich:  <br/> |Spam  <br/> |
