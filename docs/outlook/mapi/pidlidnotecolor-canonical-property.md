# <a name="pidlidnotecolor-canonical-property"></a>Kanonische PidLidNoteColor-Eigenschaft

  
  
**Betrifft**: Outlook 
  
Gibt die vorgeschlagenen Hintergrundfarbe der Notiz. 
  
|||
|:-----|:-----|
|Zugeordneten Eigenschaften:  <br/> |dispidNoteColor  <br/> |
|-Eigenschaft festgelegt:  <br/> |PSETID_Note  <br/> |
|Long-ID (Abdeckung):  <br/> |0x00008B00  <br/> |
|Datentyp:  <br/> |PT_LONG  <br/> |
|Bereich:  <br/> |Kurznotizen  <br/> |
   
## <a name="remarks"></a>Hinweise

Diese Eigenschaft muss einer der Einträge in der folgenden Tabelle werden:
  
|**Wert**|**Color**|
|:-----|:-----|
|0x00000000  <br/> |Blau  <br/> |
|0x00000001  <br/> |Green  <br/> |
|0x00000002  <br/> |Rosa  <br/> |
|0 x 00000003  <br/> |Yellow  <br/> |
|0 x 00000004  <br/> |Weiß  <br/> |
