# <a name="pidtagmessagesizeextended-canonical-property"></a>Kanonische PidTagMessageSizeExtended-Eigenschaft

  
  
**Betrifft**: Outlook 
  
Enthält eine 64-Bit-Ganzzahl-Version der **PR_MESSAGE_SIZE** ([PidTagMessageSize](pidtagmessagesize-canonical-property.md))-Eigenschaft.
  
|||
|:-----|:-----|
|Zugeordneten Eigenschaften:  <br/> |PR_MESSAGE_SIZE_EXTENDED  <br/> |
|Bezeichner:  <br/> |0x0E08  <br/> |
|Datentyp:  <br/> |PT_I8  <br/> |
|Bereich:  <br/> |Allgemeine messaging  <br/> |
