# <a name="pidlidcontactlinkentry-canonical-property"></a>Kanonische Pidlidcontactlinkentry (-Eigenschaft

  
  
**Gilt für**: Outlook 2013 | Outlook 2016 
  
Enthält die Liste der Adressbuch **EntryIDs** , die mit diesem Nachrichtenobjekt verknüpft sind. 
  
|||
|:-----|:-----|
|Zugeordnete Eigenschaften:  <br/> |dispidContactLinkEntry  <br/> |
|Eigenschaftensatz:  <br/> |PSETID_Common  <br/> |
|Long-ID (Deckel):  <br/> |0x00008585  <br/> |
|Datentyp:  <br/> |PT_BINARY  <br/> |
|Bereich:  <br/> |Kontakt  <br/> |
