# <a name="pidlidfax1originalentryid-canonical-property"></a>Kanonische Pidlidfax1originalentryid (-Eigenschaft

  
  
**Gilt für**: Outlook 2013 | Outlook 2016 
  
Gibt die ursprüngliche Eintrags-Nr. für die geschäftliche Faxnummer des Kontakts an.
  
|||
|:-----|:-----|
|Zugeordnete Eigenschaften:  <br/> |dispidFax1OriginalEntryID  <br/> |
|Eigenschaftensatz:  <br/> |PSETID_Address  <br/> |
|Long-ID (Deckel):  <br/> |0x000080B5  <br/> |
|Datentyp:  <br/> |PT_BINARY  <br/> |
|Bereich:  <br/> |Kontakt  <br/> |
   
## <a name="remarks"></a>Bemerkungen

Diese Eigenschaft, sofern vorhanden, muss die einmalige Eintrags-Nr. angeben, die dieser Faxnummer entspricht.
