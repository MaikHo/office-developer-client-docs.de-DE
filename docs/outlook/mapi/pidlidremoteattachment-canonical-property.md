# <a name="pidlidremoteattachment-canonical-property"></a>Kanonische PidLidRemoteAttachment-Eigenschaft

  
  
**Betrifft**: Outlook 
  
Gibt an, ob ein Remoteelement eine Anlage zugeordnet ist.
  
|||
|:-----|:-----|
|Zugeordneten Eigenschaften:  <br/> |dispidRemoteAttachment  <br/> |
|-Eigenschaft festgelegt:  <br/> |PSETID_Remote  <br/> |
|Long-ID (Abdeckung):  <br/> |0x00008F07  <br/> |
|Datentyp:  <br/> |PT_BOOLEAN  <br/> |
|Bereich:  <br/> |Remote-Nachricht  <br/> |
