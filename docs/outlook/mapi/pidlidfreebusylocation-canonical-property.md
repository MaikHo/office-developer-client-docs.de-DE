# <a name="pidlidfreebusylocation-canonical-property"></a>PidLidFreeBusyLocation (kanonische Eigenschaft)

  
  
**Gilt für**: Outlook 2013 | Outlook 2016 
  
Gibt einen URL-Pfad aus dem Clientidentität Frei/Gebucht-Informationen für den Kontakt als iCal-Datei als angegebenen in [[MS-OXCICAL]](https://msdn.microsoft.com/library/a685a040-5b69-4c84-b084-795113fb4012%28Office.15%29.aspx)abrufen kann.
  
|||
|:-----|:-----|
|Zugeordnete Eigenschaften:  <br/> |dispidFreeBusyLocation  <br/> |
|-Eigenschaft festgelegt:  <br/> |PSETID_Address  <br/> |
|Long-ID (Abdeckung):  <br/> |0x000080D8  <br/> |
|Datentyp:  <br/> |PT_UNICODE  <br/> |
|Bereich:  <br/> |Kontakt  <br/> |
