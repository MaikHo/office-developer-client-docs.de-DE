# <a name="pidtagformdesignername-canonical-property"></a>Kanonische PidTagFormDesignerName-Eigenschaft

  
  
**Betrifft**: Outlook 
  
Enthält den Anzeigenamen für das Objekt, das verwendet wird, um das Formular entwerfen. 
  
|||
|:-----|:-----|
|Zugeordneten Eigenschaften:  <br/> |PR_FORM_DESIGNER_NAME, PR_FORM_DESIGNER_NAME_A, PR_FORM_DESIGNER_NAME_W  <br/> |
|Bezeichner:  <br/> |0x3308  <br/> |
|Datentyp:  <br/> |PT_STRING8, PT_UNICODE  <br/> |
|Bereich:  <br/> |Allgemeine MAPI  <br/> |
   
## <a name="remarks"></a>Hinweise

Diese Eigenschaften enthalten den eindeutigen Bezeichner für das Formular-Designer-Objekt.
