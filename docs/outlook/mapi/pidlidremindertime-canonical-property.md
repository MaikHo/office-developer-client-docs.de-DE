# <a name="pidlidremindertime-canonical-property"></a>Kanonische Pidlidremindertime (-Eigenschaft

  
  
**Gilt für**: Outlook 2013 | Outlook 2016 
  
Gibt die anfängliche Signal Zeit für eine Erinnerung an.
  
|||
|:-----|:-----|
|Zugeordnete Eigenschaften:  <br/> |dispidReminderTime  <br/> |
|Eigenschaftensatz:  <br/> |PSETID_Common  <br/> |
|Long-ID (Deckel):  <br/> |0x00008502  <br/> |
|Datentyp:  <br/> |PT_SYSTIME  <br/> |
|Bereich:  <br/> |Erinnerung  <br/> |
   
## <a name="remarks"></a>Bemerkungen

Bei Calendar-Objekten stellt diese Eigenschaft die Uhrzeit dar, zu der der Benutzer zu spät kommt, wobei es sich um die Startzeit des Termins handelt. Clients müssen den Wert in koordinierter weltZeit (UTC) festlegen.
