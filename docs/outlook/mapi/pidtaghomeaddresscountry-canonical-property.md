# <a name="pidtaghomeaddresscountry-canonical-property"></a>Kanonische Pidtaghomeaddresscountry (-Eigenschaft

  
  
**Gilt für**: Outlook 2013 | Outlook 2016 
  
Enthält die Grafschaft in der Adresse eines Kontakts.
  
|||
|:-----|:-----|
|Zugeordnete Eigenschaften:  <br/> |PR_HOME_ADDRESS_COUNTRY, PR_HOME_ADDRESS_COUNTRY_A, PR_HOME_ADDRESS_COUNTRY_W  <br/> |
|Kennung:  <br/> |0x3A5A  <br/> |
|Datentyp:  <br/> |PT_STRING8, PT_UNICODE  <br/> |
|Bereich:  <br/> |MAPI-e-Mail-Benutzer  <br/> |
   
## <a name="remarks"></a>Bemerkungen

Diese Eigenschaften werden vom Benutzer oder der Organisation des Benutzers definiert.
