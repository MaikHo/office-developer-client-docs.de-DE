# <a name="pidlidtaskversion-canonical-property"></a>Kanonische PidLidTaskVersion-Eigenschaft

  
  
**Betrifft**: Outlook 
  
Gibt an, welche Kopie eines Vorgangs das neueste Update ist.
  
|||
|:-----|:-----|
|Zugeordneten Eigenschaften:  <br/> |dispidTaskVersion  <br/> |
|-Eigenschaft festgelegt:  <br/> |PSETID_Task  <br/> |
|Long-ID (Abdeckung):  <br/> |0x00008112  <br/> |
|Datentyp:  <br/> |PT_LONG  <br/> |
|Bereich:  <br/> |Aufgabe  <br/> |
   
## <a name="remarks"></a>Hinweise

Updates mit niedrigeren Versionen als die Aufgabe werden ignoriert. 
  
Wenn eine Aufgabe in einer Aufgabe Mitteilung einbetten, wird der Client die aktuelle Version der eingebetteten Aufgabe bei der Aufgabe-Kommunikation.
