# <a name="pidtagcontactaddressbookfolderentryids-canonical-property"></a>Kanonische Pidtagcontactaddressbookfolderentryids (-Eigenschaft

  
  
**Gilt für**: Outlook 2013 | Outlook 2016 
  
Enthält Bezeichner für die Kontakteordner in den Nachrichten speichern, in denen e-Mail-Benutzer aufbewahrt werden.
  
|||
|:-----|:-----|
|Zugeordnete Eigenschaften:  <br/> |PR_CONTAB_FOLDER_ENTRYIDS  <br/> |
|Kennung:  <br/> |0x6620  <br/> |
|Datentyp:  <br/> |PT_MV_BINARY  <br/> |
|Bereich:  <br/> |Kontakt Adressbuch  <br/> |
   
## <a name="remarks"></a>Bemerkungen

Dies ist eine Eigenschaft in einem Profil Abschnitt des Kontakt Adressbuchs.
