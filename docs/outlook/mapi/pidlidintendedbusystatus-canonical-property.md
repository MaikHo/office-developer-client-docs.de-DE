# <a name="pidlidintendedbusystatus-canonical-property"></a>Kanonische Pidlidintendedbusystatus (-Eigenschaft

  
  
**Gilt für**: Outlook 2013 | Outlook 2016 
  
Gibt den Wert der **dispidBusyStatus** ([pidlidbusystatus (](pidlidbusystatus-canonical-property.md))-Eigenschaft in der Besprechung im Kalender des Organisators an, als die Besprechungsanfrage oder das Besprechungs Update gesendet wurde.
  
|||
|:-----|:-----|
|Zugeordnete Eigenschaften:  <br/> |dispidIntendedBusyStatus  <br/> |
|Eigenschaftensatz:  <br/> |PSETID_Appointment  <br/> |
|Long-ID (Deckel):  <br/> |0x00008224  <br/> |
|Datentyp:  <br/> |PT_LONG  <br/> |
|Bereich:  <br/> |Meetings  <br/> |
   
## <a name="remarks"></a>Bemerkungen

Die zulässigen Werte dieser Eigenschaft entsprechen denen für die Eigenschaft **dispidBusyStatus**.
