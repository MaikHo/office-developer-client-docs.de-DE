# <a name="attreciptable"></a>attRecipTable

**Betrifft**: Outlook 2013 | Outlook 2016 
  
Wenn eine Empfänger Tabelle codiert wird, wird als einer Reihe von MAPI-Eigenschaften jeden Empfänger codiert. Das Format lautet wie folgt: 
  
_Row_Seq:_
  
>  _Anzahl der Zeilen_ _Property_Sequence_... 
    
Das Format des _Property_Seq_ ist identisch mit dem [AttMAPIProps](attmapiprops.md) -Attribut.
