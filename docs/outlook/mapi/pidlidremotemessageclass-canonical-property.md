# <a name="pidlidremotemessageclass-canonical-property"></a>Kanonische PidLidRemoteMessageClass-Eigenschaft

  
  
**Betrifft**: Outlook 
  
Gibt die Nachrichtenklasse eines remote-Elements an.
  
|||
|:-----|:-----|
|Zugeordneten Eigenschaften:  <br/> |dispidRemoteMsgClass  <br/> |
|-Eigenschaft festgelegt:  <br/> |PSETID_Remote  <br/> |
|Long-ID (Abdeckung):  <br/> |0x00008F02  <br/> |
|Datentyp:  <br/> |Keine  <br/> |
|Bereich:  <br/> |Remote-Nachricht  <br/> |
