# <a name="pidtagautoforwardcomment-canonical-property"></a>Kanonische PidTagAutoForwardComment-Eigenschaft

  
  
**Betrifft**: Outlook 
  
Enthält einen Kommentar vom Agent automatische Weiterleitung hinzugefügt.
  
|||
|:-----|:-----|
|Zugeordneten Eigenschaften:  <br/> |PR_AUTO_FORWARD_COMMENT, PR_AUTO_FORWARD_COMMENT_A, PR_AUTO_FORWARD_COMMENT_W  <br/> |
|Bezeichner:  <br/> |0x0004  <br/> |
|Datentyp:  <br/> |PT_STRING8, PT_UNICODE  <br/> |
|Bereich:  <br/> |Allgemeine Berichterstattung  <br/> |
