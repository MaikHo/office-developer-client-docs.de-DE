# <a name="pidtagspousename-canonical-property"></a>PidTagSpouseName (kanonische Eigenschaft)

  
  
**Betrifft**: Outlook 
  
Enthält den Namen des Benutzers Partner/in.
  
|||
|:-----|:-----|
|Zugeordneten Eigenschaften:  <br/> |PR_SPOUSE_NAME, PR_SPOUSE_NAME_A, PR_SPOUSE_NAME_W  <br/> |
|Bezeichner:  <br/> |0x3A48  <br/> |
|Datentyp:  <br/> |PT_STRING8, PT_UNICODE  <br/> |
|Bereich:  <br/> |MAPI-e-Mail-Benutzer  <br/> |
   
## <a name="remarks"></a>Bemerkungen

Diese Eigenschaften werden durch den Benutzer oder die Organisation des Benutzers definiert.
