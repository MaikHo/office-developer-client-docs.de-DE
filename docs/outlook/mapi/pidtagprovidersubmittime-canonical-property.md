# <a name="pidtagprovidersubmittime-canonical-property"></a>PidTagProviderSubmitTime (kanonische Eigenschaft)

  
  
**Betrifft**: Outlook 2013 | Outlook 2016 
  
Enthält Datum und Uhrzeit, ein Transportdienstes eine Nachricht an die zugrunde liegenden messaging-System übergeben.
  
|||
|:-----|:-----|
|Zugeordnete Eigenschaften:  <br/> |PR_PROVIDER_SUBMIT_TIME  <br/> |
|Kennung:  <br/> |0x0048  <br/> |
|Datentyp:  <br/> |PT_SYSTIME  <br/> |
|Bereich:  <br/> |MAPI-Umschlag  <br/> |
   
## <a name="remarks"></a>HinwBemerkungeneise

Diese Eigenschaft wird von der ausgehenden Adressbuchhierarchie zur Zeit festgelegt, die eine Nachricht gesendet wird.
  
Diese Eigenschaft entspricht einem x. 400-Übermittlung Umschlag-Message-Attribut.
