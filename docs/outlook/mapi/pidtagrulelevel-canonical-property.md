# <a name="pidtagrulelevel-canonical-property"></a>Kanonische Pidtagrulelevel (-Eigenschaft

  
  
**Gilt für**: Outlook 2013 | Outlook 2016 
  
Enthält die exitebene einer Regel.
  
|||
|:-----|:-----|
|Zugeordnete Eigenschaften:  <br/> |PR_RULE_LEVEL  <br/> |
|Kennung:  <br/> |0x6683  <br/> |
|Datentyp:  <br/> |PT_LONG  <br/> |
|Bereich:  <br/> |Server seitige Regeln  <br/> |
   
## <a name="remarks"></a>Bemerkungen

Wenn Sie diese Eigenschaft festlegen, muss der Client 0x00000000 passieren.
