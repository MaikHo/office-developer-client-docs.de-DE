# <a name="pidlidfileunder-canonical-property"></a>Kanonische Pidlidfileunder (-Eigenschaft

  
  
**Gilt für**: Outlook 2013 | Outlook 2016 
  
Gibt den Namen an, unter dem der Kontakt beim Anzeigen einer Liste von Kontakten gespeichert wird.
  
|||
|:-----|:-----|
|Zugeordnete Eigenschaften:  <br/> |dispidFileUnder  <br/> |
|Eigenschaftensatz:  <br/> |PSETID_Address  <br/> |
|Long-ID (Deckel):  <br/> |0x00008005  <br/> |
|Datentyp:  <br/> |PT_UNICODE  <br/> |
|Bereich:  <br/> |Kontakt  <br/> |
   
## <a name="remarks"></a>Bemerkungen

Die Anwendung sollte diese Eigenschaft als leere PT_UNICODE behandeln, wenn Sie im Kontakt fehlt.
