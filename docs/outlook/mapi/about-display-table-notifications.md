<span data-ttu-id="a490f-105">Benachrichtigungen für eine Anzeigetabelle werden von dem Dienstanbieter gesendet, der für die Erstellung der Anzeigetabelle in MAPI zuständig ist.</span><span class="sxs-lookup"><span data-stu-id="a490f-105">Notifications on a display table are sent by the service provider responsible for creating the display table to MAPI.</span></span> <span data-ttu-id="a490f-106">MAPI registriert diese Benachrichtigungen, indem die [IMAPITable:: Advise](imapitable-advise.md) -Methode der Display-Tabelle aufgerufen und das Table modified-Ereignis angegeben wird.</span><span class="sxs-lookup"><span data-stu-id="a490f-106">MAPI registers for these notifications by calling a display table's [IMAPITable::Advise](imapitable-advise.md) method and specifying the table modified event.</span></span>
