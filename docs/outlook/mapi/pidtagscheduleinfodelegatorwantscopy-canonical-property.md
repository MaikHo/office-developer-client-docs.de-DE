# <a name="pidtagscheduleinfodelegatorwantscopy-canonical-property"></a>Kanonische PidTagScheduleInfoDelegatorWantsCopy-Eigenschaft

  
  
**Betrifft**: Outlook 
  
Enthält True, wenn Stellvertreters möchte Kopien der Objekte bezüglich Besprechungen erhalten, die an die Stellvertretung gesendet werden.
  
|||
|:-----|:-----|
|Zugeordneten Eigenschaften:  <br/> |PR_SCHDINFO_BOSS_WANTS_COPY  <br/> |
|Bezeichner:  <br/> |0x6842  <br/> |
|Datentyp:  <br/> |PT_BOOLEAN  <br/> |
|Bereich:  <br/> |Frei/Gebucht-Informationen  <br/> |
   
## <a name="remarks"></a>Hinweise

Diese Eigenschaft muss im Delegaten Informationen-Objekt festgelegt werden.
