# <a name="pidtagoriginallyintendedrecipentryid-canonical-property"></a>Kanonische Pidtagoriginallyintendedrecipentryid (-Eigenschaft

  
  
**Gilt für**: Outlook 2013 | Outlook 2016 
  
Enthält die Eintrags-ID des ursprünglich vorgesehenen Empfängers einer automatisch weitergeleiteten Nachricht.
  
|||
|:-----|:-----|
|Zugeordnete Eigenschaften:  <br/> |PR_ORIGINALLY_INTENDED_RECIP_ENTRYID  <br/> |
|Kennung:  <br/> |0x1012  <br/> |
|Datentyp:  <br/> |PT_BINARY  <br/> |
|Bereich:  <br/> |Server  <br/> |
   
## <a name="remarks"></a>Bemerkungen

Diese Eigenschaft ist eine der Adresseigenschaften für den ursprünglich beabsichtigten Nachrichtenempfänger. Er muss vom automatischen Agent festgelegt werden, der die Nachricht weitergeleitet hat.
  
Diese Eigenschaft entspricht dem Attribut X. 400 pro Empfänger.
