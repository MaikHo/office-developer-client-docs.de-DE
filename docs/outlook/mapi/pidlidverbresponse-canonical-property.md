# <a name="pidlidverbresponse-canonical-property"></a>Kanonische Pidlidverbresponse (-Eigenschaft

  
  
**Gilt für**: Outlook 2013 | Outlook 2016 
  
Gibt die Abstimmungsoption an, die ein Teilnehmer ausgewählt hat.
  
|||
|:-----|:-----|
|Zugeordnete Eigenschaften:  <br/> |dispidVerbResponse  <br/> |
|Eigenschaftensatz:  <br/> |PSETID_Common  <br/> |
|Long-ID (Deckel):  <br/> |0x00008524  <br/> |
|Datentyp:  <br/> |PT_UNICODE  <br/> |
|Bereich:  <br/> |Allgemeine Nachrichtenübermittlung  <br/> |
   
## <a name="remarks"></a>Bemerkungen

Diese Eigenschaft wird normalerweise auf einen der Trennzeichen festgelegt, die in der **dispidVerbStream** ([pidlidverbstream (](pidlidverbstream-canonical-property.md))-Eigenschaft enthalten sind, von der der Teilnehmer abstimmt.
