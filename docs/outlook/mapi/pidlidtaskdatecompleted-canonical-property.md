# <a name="pidlidtaskdatecompleted-canonical-property"></a>Kanonische Pidlidtaskdatecompleted (-Eigenschaft

  
  
**Gilt für**: Outlook 2013 | Outlook 2016 
  
Gibt das Datum an, an dem der Benutzer die Aufgabe abgeschlossen hat.
  
|||
|:-----|:-----|
|Zugeordnete Eigenschaften:  <br/> |dispidTaskDateCompleted  <br/> |
|Eigenschaftensatz:  <br/> |PSETID_Task  <br/> |
|Long-ID (Deckel):  <br/> |0x0000810F  <br/> |
|Datentyp:  <br/> |PT_SYSTIME  <br/> |
|Bereich:  <br/> |Aufgabe  <br/> |
   
## <a name="remarks"></a>Bemerkungen

Wenn diese Eigenschaft festgelegt ist, muss Sie eine Zeitkomponente von Mitternacht in der lokalen Zeitzone aufweisen, die in UTC (Coordinated Universal Time) konvertiert wird.
