# <a name="pidtagcontentconfidentialityalgorithmid-canonical-property"></a>Kanonische Pidtagcontentconfidentialityalgorithmid (-Eigenschaft

  
  
**Gilt für**: Outlook 2013 | Outlook 2016 
  
Enthält einen Bezeichner für den Algorithmus, der zum Bestätigen der Vertraulichkeit von Nachrichteninhalten verwendet wird.
  
|||
|:-----|:-----|
|Zugeordnete Eigenschaften:  <br/> |PR_CONTENT_CONFIDENTIALITY_ALGORITHM_ID  <br/> |
|Kennung:  <br/> |0x0006  <br/> |
|Datentyp:  <br/> |PT_BINARY  <br/> |
|Bereich:  <br/> |Exchange  <br/> |
