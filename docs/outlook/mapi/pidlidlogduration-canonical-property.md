# <a name="pidlidlogduration-canonical-property"></a>Kanonische Pidlidlogduration (-Eigenschaft

  
  
**Gilt für**: Outlook 2013 | Outlook 2016 
  
Stellt die Dauer einer Journalnachricht in Minuten dar.
  
|||
|:-----|:-----|
|Zugeordnete Eigenschaften:  <br/> |dispidLogDuration  <br/> |
|Eigenschaftensatz:  <br/> |PSETID_Log  <br/> |
|Long-ID (Deckel):  <br/> |0x00008707  <br/> |
|Datentyp:  <br/> |PT_LONG  <br/> |
|Bereich:  <br/> |Journal  <br/> |
   
## <a name="remarks"></a>Bemerkungen

Die Dauer in Minuten der Aktivität, die die Differenz zwischen den Eigenschaften **dispidLogEnd** ([Pidlidlogend (](pidlidlogend-canonical-property.md)) und **dispidLogStart** ([pidlidlogstart (](pidlidlogstart-canonical-property.md)) sein muss.
