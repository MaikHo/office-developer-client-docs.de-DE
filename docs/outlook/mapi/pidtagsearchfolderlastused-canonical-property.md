# <a name="pidtagsearchfolderlastused-canonical-property"></a>Kanonische Pidtagsearchfolderlastused (-Eigenschaft

  
  
**Gilt für**: Outlook 2013 | Outlook 2016 
  
Stellt den Zeitpunkt des letzten Zugriffs auf den Ordner dar.
  
|||
|:-----|:-----|
|Zugeordnete Eigenschaften:  <br/> |PR_WB_SF_LAST_USED  <br/> |
|Kennung:  <br/> |0x6834  <br/> |
|Datentyp:  <br/> |PT_LONG  <br/> |
|Bereich:  <br/> |Suche  <br/> |
   
## <a name="remarks"></a>Bemerkungen

Diese Eigenschaft muss als Anzahl von Minuten seit Mitternacht koordinierter weltZeit (UTC) 1. Januar 1601 formatiert werden.
