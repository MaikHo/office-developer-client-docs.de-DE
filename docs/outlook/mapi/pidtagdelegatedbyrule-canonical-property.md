# <a name="pidtagdelegatedbyrule-canonical-property"></a>Kanonische Pidtagdelegatedbyrule (-Eigenschaft

  
  
**Gilt für**: Outlook 2013 | Outlook 2016 
  
Gibt an, ob die Nachricht eines Ordners von einer Regel delegiert wird.
  
|||
|:-----|:-----|
|Zugeordnete Eigenschaften:  <br/> |PR_DELEGATED_BY_RULE  <br/> |
|Kennung:  <br/> |0x3FE3  <br/> |
|Datentyp:  <br/> |PT_BOOLEAN  <br/> |
|Bereich:  <br/> |MAPI-Status  <br/> |
