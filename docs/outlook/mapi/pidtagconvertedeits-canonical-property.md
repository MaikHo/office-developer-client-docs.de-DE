# <a name="pidtagconvertedeits-canonical-property"></a>Kanonische Pidtagconvertedeits (-Eigenschaft

  
  
**Gilt für**: Outlook 2013 | Outlook 2016 
  
Enthält einen Bezeichner für die Typen von Text in einer Nachricht nach der Konvertierung.
  
|||
|:-----|:-----|
|Zugeordnete Eigenschaften:  <br/> |PR_CONVERTED_EITS  <br/> |
|Kennung:  <br/> |0x000E  <br/> |
|Datentyp:  <br/> |PT_BINARY  <br/> |
|Bereich:  <br/> |Exchange  <br/> |
   
## <a name="remarks"></a>Bemerkungen

Diese Eigenschaft gibt an, welche codierten Informationstypen (EITs) zum Konvertieren der Textkomponenten der Nachricht verwendet wurden.
