# <a name="pidtagpreferredbyname-canonical-property"></a>PidTagPreferredByName (kanonische Eigenschaft)

  
  
**Betrifft**: Outlook 
  
Enthält den Namen, den der Empfänger bevorzugt aufgerufen werden.
  
|||
|:-----|:-----|
|Zugeordneten Eigenschaften:  <br/> |PR_PREFERRED_BY_NAME, PR_PREFERRED_BY_NAME_A, PR_PREFERRED_BY_NAME_W  <br/> |
|Bezeichner:  <br/> |0x3A47  <br/> |
|Datentyp:  <br/> |PT_STRING8, PT_UNICODE  <br/> |
|Bereich:  <br/> |MAPI-e-Mail-Benutzer  <br/> |
