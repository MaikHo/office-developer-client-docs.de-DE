# <a name="pidtagnonipmsubtreeentryid-canonical-property"></a>Kanonische Pidtagnonipmsubtreeentryid (-Eigenschaft

  
  
**Gilt für**: Outlook 2013 | Outlook 2016 
  
Zeigt die Eintrags-ID des **NON_IPM_SUBTREE** -Ordners in einem Exchange-informationsSpeicher für Öffentliche Ordner an. 
  
|||
|:-----|:-----|
|Zugeordnete Eigenschaften:  <br/> |PR_NON_IPM_SUBTREE_ENTRYID  <br/> |
|Kennung:  <br/> |0x6620  <br/> |
|Datentyp:  <br/> |PT_BINARY  <br/> |
|Bereich:  <br/> |Nachrichtenspeicher  <br/> |
