# <a name="install-mapi-header-files"></a>Installieren der MAPI-Headerdateien

**Gilt für**: Outlook 2013 | Outlook 2016 
  
In diesem Thema wird beschrieben, wie Sie MAPI-Headerdateien und Bibliotheksdateien abrufen und installieren.

MAPI-Headerdateien stehen als Download unter [Outlook 2010: MAPI-Header-Dateien](https://www.microsoft.com/downloads/details.aspx?FamilyID=f8d01fc8-f7b5-4228-baa3-817488a66db1)zur Verfügung. MAPI-Headerdateien werden standardmäßig im folgenden Ordner installiert: „C:\Program Files\Microsoft SDKs\Office\14.0\Include“.
  
Eine Liste der MAPI-Header-Dateien finden Sie unter [Häufig verwendete Headerdateien](commonly-used-header-files.md).
