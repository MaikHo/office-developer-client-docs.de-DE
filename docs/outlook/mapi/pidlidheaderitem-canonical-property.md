# <a name="pidlidheaderitem-canonical-property"></a>Kanonische PidLidHeaderItem-Eigenschaft

  
  
**Betrifft**: Outlook 
  
Gibt an, ob Microsoft Outlook auf dem Clientcomputer heruntergeladen wurde nur die Kopfzeile einer Nachricht oder Überschrift und Text der Nachricht, wenn Outlook im Exchange-Cache-Modus konfiguriert ist.
  
|||
|:-----|:-----|
|Zugeordneten Eigenschaften:  <br/> |dispidHeaderItem  <br/> |
|-Eigenschaft festgelegt:  <br/> |PSETID_Common  <br/> |
|Long-ID (Abdeckung):  <br/> |0x00008578  <br/> |
|Datentyp:  <br/> |PT_LONG  <br/> |
|Bereich:  <br/> |Allgemeine messaging  <br/> |
