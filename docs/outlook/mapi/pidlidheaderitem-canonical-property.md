# <a name="pidlidheaderitem-canonical-property"></a>Kanonische Pidlidheaderitem (-Eigenschaft

  
  
**Gilt für**: Outlook 2013 | Outlook 2016 
  
Gibt an, ob Microsoft Outlook nur die Kopfzeile einer Nachricht oder sowohl die Kopfzeile als auch den Text der Nachricht auf den Clientcomputer heruntergeladen hat, wenn Outlook im Exchange-Cache-Modus konfiguriert ist.
  
|||
|:-----|:-----|
|Zugeordnete Eigenschaften:  <br/> |dispidHeaderItem  <br/> |
|Eigenschaftensatz:  <br/> |PSETID_Common  <br/> |
|Long-ID (Deckel):  <br/> |0x00008578  <br/> |
|Datentyp:  <br/> |PT_LONG  <br/> |
|Bereich:  <br/> |Allgemeine Nachrichtenübermittlung  <br/> |
