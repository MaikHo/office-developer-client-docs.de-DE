# <a name="operating-environment-issues"></a>Probleme in der Betriebssystemumgebung

  
  
**Betrifft**: Outlook 2013 | Outlook 2016 
  
Die Aspekte des Betriebssystems und Betriebssystem sind wichtig, wenn MAPI-Entwickler wählen Sie eine Entwicklungsumgebung und einer Umgebung für die Installation von Windows SDK.
  
MAPI-Entwickler müssen beachten Sie, dass diese Optionen wirken sich auf das Betriebssystem, das auf ihrer Komponente ausgeführt wird und der Bereich der Komponenten, die als Ziel werden.
