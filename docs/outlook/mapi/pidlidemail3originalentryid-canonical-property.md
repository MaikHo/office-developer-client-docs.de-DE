# <a name="pidlidemail3originalentryid-canonical-property"></a>Kanonische Pidlidemail3originalentryid (-Eigenschaft

  
  
**Gilt für**: Outlook 2013 | Outlook 2016 
  
Gibt die **Eintrags** -Nr. des Objekts an, das der dritten e-Mail-Adresse entspricht. 
  
|||
|:-----|:-----|
|Zugeordnete Eigenschaften:  <br/> |dispidEmail3OriginalEntryID  <br/> |
|Eigenschaftensatz:  <br/> |PSETID_Address  <br/> |
|Long-ID (Deckel):  <br/> |0x000080A5  <br/> |
|Datentyp:  <br/> |PT_BINARY  <br/> |
|Bereich:  <br/> |Kontakt  <br/> |
   
## <a name="remarks"></a>Bemerkungen

Bei dem Wert dieser Eigenschaft muss es sich um eine einmalige **Eingabe** -e-Adresse oder ein gültiges Adressbuchobjekt- **Eintrags**-Nr handeln.
