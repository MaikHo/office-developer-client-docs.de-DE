# <a name="pidtagproofofsubmission-canonical-property"></a>Kanonische Pidtagproofofsubmission (-Eigenschaft

  
  
**Gilt für**: Outlook 2013 | Outlook 2016 
  
Diese Eigenschaft enthält TRUE, wenn ein Nachrichtenabsender den Nachweis anfordert, dass das Nachrichtenübertragungssystem eine Nachricht zur Übermittlung an den ursprünglich vorgesehenen Empfänger gesendet hat.
  
|||
|:-----|:-----|
|Zugeordnete Eigenschaften:  <br/> |PR_PROOF_OF_SUBMISSION  <br/> |
|Kennung:  <br/> |0x0E26  <br/> |
|Datentyp:  <br/> |PT_BINARY  <br/> |
|Bereich:  <br/> |Nicht transmitable MAPI  <br/> |
