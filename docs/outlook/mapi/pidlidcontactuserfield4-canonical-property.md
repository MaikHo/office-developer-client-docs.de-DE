# <a name="pidlidcontactuserfield4-canonical-property"></a>Kanonische pidlidcontactuserfield4 (-Eigenschaft

  
  
**Gilt für**: Outlook 2013 | Outlook 2016 
  
Gibt das vierte Feld für den Kontakt an, das für verschiedene Zwecke für den Kontakt vorgesehen ist.
  
|||
|:-----|:-----|
|Zugeordnete Eigenschaften:  <br/> |dispidContactUserField4  <br/> |
|Eigenschaftengruppe:  <br/> |PSETID_Address  <br/> |
|Lange ID (LID):  <br/> |0x00008052  <br/> |
|Datentyp:  <br/> |PT_UNICODE  <br/> |
|Bereich:  <br/> |Kontakt  <br/> |
   
## <a name="remarks"></a>Hinweise

Diese Eigenschaft enthält Text, der nicht mit einer anderen Kontakt spezifischen Eigenschaft in Beziehung steht.
