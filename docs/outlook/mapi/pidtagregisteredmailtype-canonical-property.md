# <a name="pidtagregisteredmailtype-canonical-property"></a>Kanonische Pidtagregisteredmailtype (-Eigenschaft

  
  
**Gilt für**: Outlook 2013 | Outlook 2016 
  
Diese Eigenschaft enthält den Registrierungstyp, der für die physische Übermittlung einer Nachricht verwendet wird.
  
|||
|:-----|:-----|
|Zugeordnete Eigenschaften:  <br/> |PR_REGISTERED_MAIL_TYPE  <br/> |
|Kennung:  <br/> |0x0C16  <br/> |
|Datentyp:  <br/> |PT_LONG  <br/> |
|Bereich:  <br/> |MAPI-Empfänger  <br/> |
