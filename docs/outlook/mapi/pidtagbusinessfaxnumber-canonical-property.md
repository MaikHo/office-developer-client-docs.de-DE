# <a name="pidtagbusinessfaxnumber-canonical-property"></a>Kanonische Pidtagbusinessfaxnumber (-Eigenschaft

  
  
**Gilt für**: Outlook 2013 | Outlook 2016 
  
Enthält die Telefonnummer des Geschäfts Faxgeräts des Empfängers. 
  
|||
|:-----|:-----|
|Zugeordnete Eigenschaften:  <br/> |PR_BUSINESS_FAX_NUMBER, PR_BUSINESS_FAX_NUMBER_A, PR_BUSINESS_FAX_NUMBER_W  <br/> |
|Kennung:  <br/> |0x3A24  <br/> |
|Datentyp:  <br/> |PT_UNICODE, PT_STRING8  <br/> |
|Bereich:  <br/> |Kontakt  <br/> |
   
## <a name="remarks"></a>Bemerkungen

Diese Eigenschaften sind Beispiele für die Eigenschaften, die die Identifizierung und den Zugriff auf Informationen zu einem Empfänger ermöglichen. Diese Eigenschaften werden vom Empfänger und der Organisation des Empfängers definiert.
