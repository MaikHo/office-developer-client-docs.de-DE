# <a name="pidtagcomputernetworkname-canonical-property"></a>Kanonische PidTagComputerNetworkName-Eigenschaft

  
  
**Betrifft**: Outlook 
  
Enthält den Namen des Netzwerks zum Übertragen der Nachricht verwendet.
  
|||
|:-----|:-----|
|Zugeordneten Eigenschaften:  <br/> |PR_COMPUTER_NETWORK_NAME, PR_COMPUTER_NETWORK_NAME_A, PR_COMPUTER_NETWORK_NAME_W  <br/> |
|Bezeichner:  <br/> |0x3A49  <br/> |
|Datentyp:  <br/> |PT_UNICODE PT_STRING8  <br/> |
|Bereich:  <br/> |Kontakt  <br/> |
