|||
|:-----|:-----|
|Zugeordnete Eigenschaften:  <br/> |dispidRemoteEID  <br/> |
|Eigenschaftensatz:  <br/> |PSETID_Remote  <br/> |

|Datentyp:  <br/> |PT_BINARY  <br/> |
