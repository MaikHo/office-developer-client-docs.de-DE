# <a name="pidlidappointmentproposalnumber-canonical-property"></a>Kanonische pidlidappointmentproposalnumber (-Eigenschaft

  
  
**Gilt für**: Outlook 2013 | Outlook 2016 
  
Gibt die Anzahl der Teilnehmer an, die gegen Vorschläge gesendet haben, die vom Organisator nicht akzeptiert oder abgelehnt wurden.
  
|||
|:-----|:-----|
|Zugeordnete Eigenschaften:  <br/> |dispidApptProposalNum  <br/> |
|Eigenschaftengruppe:  <br/> |PSETID_Appointment  <br/> |
|Lange ID (LID):  <br/> |0x00008259  <br/> |
|Datentyp:  <br/> |PT_LONG  <br/> |
|Bereich:  <br/> |Meetings  <br/> |
