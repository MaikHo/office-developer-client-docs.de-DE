# <a name="pidnameattachmentmacinfo-canonical-property"></a>Kanonische Pidnameattachmentmacinfo (-Eigenschaft

  
  
**Gilt für**: Outlook 2013 | Outlook 2016 
  
Enthält einen [RFC3282] anlagenwert, der aus geeigneten Header-und Ressourcen Verzweigungs Daten besteht, die von Macintosh-basierten e-Mail-Clients verwendet werden.
  
|||
|:-----|:-----|
|Angezeigte Namen:  <br/> |Keine  <br/> |
|Eigenschaftensatz:  <br/> |PSETID_Attachment  <br/> |
|Eigenschaftsname:  <br/> |AttachmentMacInfo  <br/> |
|Datentyp:  <br/> |PT_BINARY  <br/> |
|Bereich:  <br/> |Nachrichtenanlage  <br/> |
   
## <a name="remarks"></a>Bemerkungen

Weitere Informationen finden Sie unter MS-OXCMAIL section 2.2.4.2 Apple File Formats.
