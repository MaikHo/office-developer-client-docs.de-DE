# <a name="pidtagbusinesshomepage-canonical-property"></a>Kanonische PidTagBusinessHomePage-Eigenschaft

  
  
**Betrifft**: Outlook 
  
Enthält die URL der Startseite für das Unternehmen.
  
|||
|:-----|:-----|
|Zugeordneten Eigenschaften:  <br/> |PR_BUSINESS_HOME_PAGE, PR_BUSINESS_HOME_PAGE_A, PR_BUSINESS_HOME_PAGE_W  <br/> |
|Bezeichner:  <br/> |0x3A51  <br/> |
|Datentyp:  <br/> |PT_UNICODE PT_STRING8  <br/> |
|Bereich:  <br/> |Kontakt  <br/> |
   
## <a name="remarks"></a>Hinweise

Diese Eigenschaften werden als ein Link zur Homepage für den Kontakt Business verwendet.
