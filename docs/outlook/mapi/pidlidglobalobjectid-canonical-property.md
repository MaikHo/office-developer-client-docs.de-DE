# <a name="pidlidglobalobjectid-canonical-property"></a>Kanonische Pidlidglobalobjectid (-Eigenschaft

  
  
**Gilt für**: Outlook 2013 | Outlook 2016 
  
Gibt den eindeutigen Bezeichner des Calendar-Objekts an.
  
|||
|:-----|:-----|
|Zugeordnete Eigenschaften:  <br/> |LID_GLOBAL_OBJID  <br/> |
|Eigenschaftensatz:  <br/> |PSETID_Meeting  <br/> |
|Long-ID (Deckel):  <br/> |0x00000003  <br/> |
|Datentyp:  <br/> |PT_BINARY  <br/> |
|Bereich:  <br/> |Meetings  <br/> |
   
## <a name="remarks"></a>Bemerkungen

Sobald für ein Calendar-Objekt festgelegt, darf der Wert dieser Eigenschaft nicht geändert werden. Eine ausführliche Beschreibung des Formats finden Sie unter [[MS-OXOCAL]](https://msdn.microsoft.com/library/9b636532-9150-4836-9635-9c9b756c9ccf%28Office.15%29.aspx).
