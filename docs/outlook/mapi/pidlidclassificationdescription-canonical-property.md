# <a name="pidlidclassificationdescription-canonical-property"></a>Kanonische pidlidclassificationdescription (-Eigenschaft

  
  
**Gilt für**: Outlook 2013 | Outlook 2016 
  
Beschreibt, was die Nachrichtenklassifikation erreichen möchte.
  
|||
|:-----|:-----|
|Zugeordnete Eigenschaften:  <br/> |dispidClassDesc  <br/> |
|Eigenschaftengruppe:  <br/> |PSETID_Common  <br/> |
|Lange ID (LID):  <br/> |0x000085B7  <br/> |
|Datentyp:  <br/> |PT_UNICODE  <br/> |
|Bereich:  <br/> |Allgemeines Messaging  <br/> |
