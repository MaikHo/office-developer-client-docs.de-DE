# <a name="pidtagexpirytime-canonical-property"></a>Kanonische Pidtagexpirytime (-Eigenschaft

  
  
**Gilt für**: Outlook 2013 | Outlook 2016 
  
Enthält das Datum und die Uhrzeit, zu denen das Messagingsystem den Inhalt einer Nachricht ungültig machen kann. 
  
|||
|:-----|:-----|
|Zugeordnete Eigenschaften:  <br/> |PR_EXPIRY_TIME  <br/> |
|Kennung:  <br/> |0x0015  <br/> |
|Datentyp:  <br/> |PT_SYSTIME  <br/> |
|Bereich:  <br/> |MAPI-Umschlag  <br/> |
   
## <a name="remarks"></a>Bemerkungen

Diese Eigenschaft wird verwendet, um das Messagingsystem bei der Verarbeitung der übermittelten zwischenmenschlichen Nachrichten zu leiten.
