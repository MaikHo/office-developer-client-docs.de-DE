# <a name="pidtagproofofsubmissionrequested-canonical-property"></a>Kanonische Pidtagproofofsubmissionrequested (-Eigenschaft

  
  
**Gilt für**: Outlook 2013 | Outlook 2016 
  
Enthält einen ASN. 1-Nachweis über die Übermittlung.
  
|||
|:-----|:-----|
|Zugeordnete Eigenschaften:  <br/> |PR_PROOF_OF_SUBMISSION_REQUESTED  <br/> |
|Kennung:  <br/> |0x0028  <br/> |
|Datentyp:  <br/> |PT_BOOLEAN  <br/> |
|Bereich:  <br/> |MAPI-Umschlag  <br/> |
   
## <a name="remarks"></a>Bemerkungen

Diese Eigenschaft entspricht einem X. 400-Übermittlungs Umschlag pro Nachrichtenattribut.
