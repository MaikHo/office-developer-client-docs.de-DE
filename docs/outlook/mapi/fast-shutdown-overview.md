# <a name="fast-shutdown-overview"></a><span data-ttu-id="7b65b-103">Übersicht über das schnelle Herunterfahren</span><span class="sxs-lookup"><span data-stu-id="7b65b-103">Fast Shutdown Overview</span></span>

<span data-ttu-id="7b65b-104">**Gilt für**: Outlook 2013 | Outlook 2016</span><span class="sxs-lookup"><span data-stu-id="7b65b-104">**Applies to**: Outlook 2013 | Outlook 2016</span></span> 
  
<span data-ttu-id="7b65b-105">Das schnelle Herunterfahren ist ein Mechanismus für einen MAPI-Client, um ein schnelles Herunterfahren des Clientprozesses zu initiieren, wobei alle Anbieter benachrichtigt werden, mit denen der Client über eine aktive MAPI-Sitzung verfügt, um Daten und Einstellungen zu speichern, bevor der Clientprozess beendet wird.</span><span class="sxs-lookup"><span data-stu-id="7b65b-105">Fast shutdown is a mechanism for a MAPI client to initiate a quick shutdown of the client process, notifying all providers with which the client has an active MAPI session to save data and settings before the client process exits.</span></span> <span data-ttu-id="7b65b-106">In diesem Thema wird der grundlegende Mechanismus für schnelles Herunterfahren beschrieben.</span><span class="sxs-lookup"><span data-stu-id="7b65b-106">This topic describes the basic mechanism of fast shutdown.</span></span> 

<span data-ttu-id="7b65b-107">Beginnend mit Microsoft Outlook 2010 und jetzt einschließlich Microsoft Outlook 2013, stellt das MAPI-Subsystem die [IMAPIClientShutdown: IUnknown](imapiclientshutdowniunknown.md) -Schnittstelle bereit.</span><span class="sxs-lookup"><span data-stu-id="7b65b-107">Starting in Microsoft Outlook 2010 and now including Microsoft Outlook 2013, the MAPI subsystem provides the [IMAPIClientShutdown : IUnknown](imapiclientshutdowniunknown.md) interface.</span></span> <span data-ttu-id="7b65b-108">Outlook und andere MAPI-Clients können das schnelle Herunterfahren als Standardmechanismus zum Beenden des Clientprozesses festlegen.</span><span class="sxs-lookup"><span data-stu-id="7b65b-108">Outlook and other MAPI clients can adopt fast shutdown as the default mechanism to exit the client process.</span></span> <span data-ttu-id="7b65b-109">Eine Einstellung auf Benutzerebene in der Windows-Registrierung des Clientcomputers steuert das schnelle Herunterfahren für alle MAPI-Clients für diesen Benutzer auf diesem Computer.</span><span class="sxs-lookup"><span data-stu-id="7b65b-109">A user-level setting in the Windows registry of the client computer controls the adoption of fast shutdown for all MAPI clients for that user on that computer.</span></span> <span data-ttu-id="7b65b-110">Ausführliche Informationen zu den Registrierungseinstellungen finden Sie unter [Benutzeroptionen für das schnelle Herunterfahren](fast-shutdown-user-options.md).</span><span class="sxs-lookup"><span data-stu-id="7b65b-110">For details about the registry settings, see [Fast Shutdown User Options](fast-shutdown-user-options.md).</span></span>
  
<span data-ttu-id="7b65b-111">Wenn ein MAPI-Client Schnelles Herunterfahren übernehmen muss, muss die **IMAPIClientShutdown: IUnknown** -Schnittstelle verwendet werden.</span><span class="sxs-lookup"><span data-stu-id="7b65b-111">If a MAPI client needs to adopt fast shutdown, it must use the **IMAPIClientShutdown : IUnknown** interface.</span></span> <span data-ttu-id="7b65b-112">Es folgt ein typischer Verlauf von Ereignissen, wenn der Client versucht, das Herunterfahren zu beenden:</span><span class="sxs-lookup"><span data-stu-id="7b65b-112">The following is the typical course of events when the client attempts to shut down:</span></span> 
  
1. <span data-ttu-id="7b65b-113">Der MAPI-Client initiiert das Herunterfahren durch Aufrufen der [IMAPIClientShutdown:: QueryFastShutdown](imapiclientshutdown-queryfastshutdown.md) -Methode, um zu bestimmen, ob das MAPI-Subsystem das schnelle Herunterfahren unterstützt.</span><span class="sxs-lookup"><span data-stu-id="7b65b-113">The MAPI client initiates the shutdown by calling the [IMAPIClientShutdown::QueryFastShutdown](imapiclientshutdown-queryfastshutdown.md) method to determine whether the MAPI subsystem supports fast shutdown.</span></span> 
    
2. <span data-ttu-id="7b65b-114">Das MAPI-Subsystem antwortet mit der verfügbaren Unterstützung für das schnelle Herunterfahren auf den **IMAPIClientShutdown:: QueryFastShutdown** -Aufruf des Clients, indem Sie das folgende Verfahren ausführen:</span><span class="sxs-lookup"><span data-stu-id="7b65b-114">The MAPI subsystem responds with the available fast shutdown support to the client's **IMAPIClientShutdown::QueryFastShutdown** call by using the following procedure:</span></span> 
    
    1. <span data-ttu-id="7b65b-115">Das MAPI-Subsystem Ruft die [IMAPIProviderShutdown:: QueryFastShutdown](imapiprovidershutdown-queryfastshutdown.md) -Methode für jeden MAPI-Anbieter auf, mit dem der MAPI-Clientprozess eine aktive MAPI-Sitzung hat, wenn der Anbieter das IMAPIProviderShutdown implementiert hat [: IUnknown](imapiprovidershutdowniunknown.md) Oberfläche.</span><span class="sxs-lookup"><span data-stu-id="7b65b-115">The MAPI subsystem calls the [IMAPIProviderShutdown::QueryFastShutdown](imapiprovidershutdown-queryfastshutdown.md) method for each MAPI provider with which the MAPI client process has an active MAPI session, if the provider has implemented the [IMAPIProviderShutdown : IUnknown](imapiprovidershutdowniunknown.md) interface.</span></span> 
        
       > [!NOTE]
       >  <span data-ttu-id="7b65b-116">Das MAPI-Subsystem fragt MAPI-Anbieter immer über die **IMAPIProviderShutdown: IUnknown** -Schnittstelle innerhalb jeder MAPI-Sitzung ab und benachrichtigt Sie in der folgenden Reihenfolge:</span><span class="sxs-lookup"><span data-stu-id="7b65b-116">The MAPI subsystem always queries and notifies MAPI providers through the **IMAPIProviderShutdown : IUnknown** interface within each MAPI session in the following order:</span></span>
       > 1. <span data-ttu-id="7b65b-117">Transport Anbieter</span><span class="sxs-lookup"><span data-stu-id="7b65b-117">Transport providers</span></span>
       > 2. <span data-ttu-id="7b65b-118">Adressbuchanbieter</span><span class="sxs-lookup"><span data-stu-id="7b65b-118">Address book providers</span></span>
       > 3. <span data-ttu-id="7b65b-119">Speicheranbieter</span><span class="sxs-lookup"><span data-stu-id="7b65b-119">Store providers</span></span> 
    
    2. <span data-ttu-id="7b65b-120">Abhängig von der Registrierungseinstellung für das schnelle Herunterfahren für diesen Benutzer auf dem Clientcomputer gibt das MAPI-Subsystem den entsprechenden Rückgabecode für **IMAPIClientShutdown:: QueryFastShutdown**an.</span><span class="sxs-lookup"><span data-stu-id="7b65b-120">Depending on the fast shutdown registry setting for that user on the client computer, the MAPI subsystem specifies the appropriate return code to **IMAPIClientShutdown::QueryFastShutdown**.</span></span> <span data-ttu-id="7b65b-121">Der Rückgabecode ist entweder S_OK oder MAPI_E_NO_SUPPORT.</span><span class="sxs-lookup"><span data-stu-id="7b65b-121">The return code is either S_OK or MAPI_E_NO_SUPPORT.</span></span>
        
    3. <span data-ttu-id="7b65b-122">Der MAPI-Client ruft die [IMAPIClientShutdown:: NotifyProcessShutdown](imapiclientshutdown-notifyprocessshutdown.md) -Methode auf, um dem MAPI-Subsystem anzuzeigen, dass das Herunterfahren beabsichtigt ist.</span><span class="sxs-lookup"><span data-stu-id="7b65b-122">The MAPI client calls the [IMAPIClientShutdown::NotifyProcessShutdown](imapiclientshutdown-notifyprocessshutdown.md) method to indicate to the MAPI subsystem the intention to shut down.</span></span> 
        
    4. <span data-ttu-id="7b65b-123">Das MAPI-Subsystem gibt jedem geladenen MAPI-Anbieter an, dass der MAPI-Client heruntergefahren wird.</span><span class="sxs-lookup"><span data-stu-id="7b65b-123">The MAPI subsystem indicates to each loaded MAPI provider that the MAPI client will shut down.</span></span> <span data-ttu-id="7b65b-124">Für die Anbieter, die die **IMAPIProviderShutdown: IUnknown** -Schnittstelle implementiert haben, ruft das MAPI-Subsystem die entsprechende [IMAPIProviderShutdown:: NotifyProcessShutdown](imapiprovidershutdown-notifyprocessshutdown.md) -Methode auf.</span><span class="sxs-lookup"><span data-stu-id="7b65b-124">For those providers that have implemented the **IMAPIProviderShutdown : IUnknown** interface, the MAPI subsystem calls the corresponding [IMAPIProviderShutdown::NotifyProcessShutdown](imapiprovidershutdown-notifyprocessshutdown.md) method.</span></span> 
        
    5. <span data-ttu-id="7b65b-125">Der MAPI-Client ruft die [IMAPIClientShutdown::D ofastshutdown](imapiclientshutdown-dofastshutdown.md) -Methode auf, um dem MAPI-Subsystem anzuzeigen, dass der Clientprozess sofort beendet wird.</span><span class="sxs-lookup"><span data-stu-id="7b65b-125">The MAPI client calls the [IMAPIClientShutdown::DoFastShutdown](imapiclientshutdown-dofastshutdown.md) method to indicate to the MAPI subsystem that the client process is exiting immediately.</span></span> 
        
    6. <span data-ttu-id="7b65b-126">Das MAPI-Subsystem gibt jedem geladenen MAPI-Anbieter an, dass der MAPI-Clientprozess beendet wird.</span><span class="sxs-lookup"><span data-stu-id="7b65b-126">The MAPI subsystem indicates to each loaded MAPI provider that the MAPI client process is exiting.</span></span> <span data-ttu-id="7b65b-127">Für die Anbieter, die die **IMAPIProviderShutdown: IUnknown** -Schnittstelle implementiert haben, ruft das MAPI-Subsystem die entsprechende [IMAPIProviderShutdown::D Ofastshutdown](imapiprovidershutdown-dofastshutdown.md) -Methode auf.</span><span class="sxs-lookup"><span data-stu-id="7b65b-127">For those providers that have implemented the **IMAPIProviderShutdown : IUnknown** interface, the MAPI subsystem calls the corresponding [IMAPIProviderShutdown::DoFastShutdown](imapiprovidershutdown-dofastshutdown.md) method.</span></span> <span data-ttu-id="7b65b-128">An diesem Punkt sollten diese MAPI-Anbieter überprüfen, ob alle erforderlichen Aktionen, wie das Speichern von Daten und Einstellungen, abgeschlossen sind, damit der MAPI-Client sofort alle Verweise und die Beendigung der Verbindung trennen kann.</span><span class="sxs-lookup"><span data-stu-id="7b65b-128">At this point, these MAPI providers should verify that all necessary actions, such as saving data and settings, are complete in preparation for the MAPI client to immediately disconnect all references and exit.</span></span>
