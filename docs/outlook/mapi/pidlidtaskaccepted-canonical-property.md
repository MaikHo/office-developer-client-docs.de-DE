# <a name="pidlidtaskaccepted-canonical-property"></a>Kanonische Pidlidtaskaccepted (-Eigenschaft

  
  
**Gilt für**: Outlook 2013 | Outlook 2016 
  
Gibt an, ob ein Aufgabenempfänger auf eine Aufgabenanfrage geantwortet hat.
  
|||
|:-----|:-----|
|Zugeordnete Eigenschaften:  <br/> |dispidTaskAccepted  <br/> |
|Eigenschaftensatz:  <br/> |PSETID_Task  <br/> |
|Long-ID (Deckel):  <br/> |0x00008108  <br/> |
|Datentyp:  <br/> |PT_BOOLEAN  <br/> |
|Bereich:  <br/> |Aufgabe  <br/> |
   
## <a name="remarks"></a>Bemerkungen

Der Client legt diese Eigenschaft für eine neue Aufgabe auf FALSE fest, oder der Client legt diese Eigenschaft auf TRUE fest, wenn ein Vorgang entweder akzeptiert oder abgelehnt wird. Wenn die Eigenschaft nicht festgelegt ist, wird der Wert FALSE angenommen.
