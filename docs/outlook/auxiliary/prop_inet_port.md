# <a name="propinetport"></a>PROP_INET_PORT

Stellt die Nummer des Anschlusses für ein allgemeines Internet Postfach dar.
  
## <a name="quick-info"></a>QuickInfo

|||
|:-----|:-----|
|Kennung:  <br/> |0x0104  <br/> |
|Eigenschafts:  <br/> |PT_DWORD  <br/> |
|Property-Tag:  <br/> |0x01040003  <br/> |
|Access  <br/> |Schreibgeschützt  <br/> |
   
## <a name="remarks"></a>Bemerkungen

Ein Beispiel ist 110 für POP-Accounts.
