# <a name="osc-sample-templates"></a>OSC-Beispielvorlagen

Outlook Social Connector 2013 Verweis auf den begleitenden sind mehrere Anbietervorlagen in C++, c# und Visual Basic. Sie können die Vorlagen auf herunterladen [Outlook Social Connector 2013: Anbietervorlagen](http://code.msdn.microsoft.com/Outlook-Social-Connector-73fd8d2c).
  
Die Anbietervorlagen vom sind für die Verwendung verfügbar, wenn Sie einen Anbieter in der Sprache Ihrer Wahl erstellen. Um die Downloadgröße des Anbieters zu minimieren und Anbieter des Ladens zu minimieren, sollten Sie einen Anbieter für Produktion mithilfe von C++ erstellen. Wenn Sie einen Anbieter für Benutzer, die .NET Framework 3.5 SP1 oder eine höhere Version von .NET Framework auf ihren Computern installiert haben entwickeln, können Sie auch berücksichtigen Schreiben eines verwalteten Anbieters mit c# oder Visual Basic.
