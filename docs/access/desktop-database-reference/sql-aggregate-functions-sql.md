# <a name="sql-aggregate-functions-sql"></a>SQL-Aggregatfunktionen (SQL)


**Betrifft**: Access 2013, Office 2013

Mithilfe der SQL-Aggregatfunktionen können Sie verschiedene Statistiken zu Wertgruppen ermitteln. Sie können diese Funktionen in einer Abfrage und in Aggregatausdrücken in der **SQL** -Eigenschaft eines **QueryDef** -Objekts verwenden, wenn Sie ein **Recordset** -Objekt basierend auf einer SQL-Abfrage erstellen.

- [AVG-Funktion](https://msdn.microsoft.com/library/ff822755\(v=office.15\))

- [Count-Funktion](https://msdn.microsoft.com/library/ff844748\(v=office.15\))

- [First-und Last](https://msdn.microsoft.com/library/ff197381\(v=office.15\))

- [Min-Funktion und Max-Funktion](https://msdn.microsoft.com/library/ff194490\(v=office.15\))

- [StDev-Funktion, StDevP-Funktion](https://msdn.microsoft.com/library/ff197043\(v=office.15\))

- [SUM-Funktion](https://msdn.microsoft.com/library/ff844764\(v=office.15\))

- [Var-Funktion, VarP-Funktion](https://msdn.microsoft.com/library/ff192105\(v=office.15\))
