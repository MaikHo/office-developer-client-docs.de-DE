# <a name="understanding-recordset-structure"></a>Grundlegendes zur Recordsetstruktur


**Betrifft**: Access 2013 | Office 2013

Jedes **Recordset** -Objekt weist eine **Fields** -Auflistung auf, die aus mindestens einem **Field** -Objekt besteht. Ein **Field** -Objekt stellt in der Regel eine Tabellenspalte dar. In den folgenden Themen wird erläutert, wie Sie in der **Fields** -Auflistung navigieren und Informationen zu jedem Feld abrufen. Anschließend wird beschrieben, welche Informationen Ihnen über das **Field** -Objekt zur Verfügung stehen und wie Sie diese verwenden.

  - [Fields-Auflistung](the-fields-collection.md)

  - [Field-Objekt](the-field-object.md)

  - [Verwenden von Recordsets](working-with-recordsets.md)
