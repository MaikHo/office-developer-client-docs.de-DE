# <a name="propertiescount-property-dao"></a>Properties. Count-Eigenschaft (DAO)


**Gilt für**: Access 2013, Office 2013

Mithilfe der **Count**-Eigenschaft können Sie die Anzahl von Elementen in der angegebenen Auflistung ermitteln. Schreibgeschützter **Long**-Wert.

## <a name="syntax"></a>Syntax

*Ausdruck* . Count

*Ausdruck* Eine Variable, die ein **Properties** -Objekt darstellt.
