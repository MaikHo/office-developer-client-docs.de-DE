# <a name="programming-with-ado-md"></a>Programmieren mit ADO MD


**Betrifft**: Access 2013, Office 2013

Zur Verwendung von ADO MD mit einem Entwicklungstool sollten Sie einen Verweis zur ADO MD-Typbibliothek einrichten. Die Beschreibung der ADO MD-Bibliothek finden Sie unter "Microsoft ActiveX Data Objects (Multi-dimensional) Library". Der Dateiname der ADO MD-Bibliothek ist msadomd.dll. Die Programm-ID (ProgID) lautet "ADOMD". Weitere Informationen zum Erstellen von Verweisen zu Bibliotheken finden Sie in der Dokumentation Ihres Entwicklungstools.
