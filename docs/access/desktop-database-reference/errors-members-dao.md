# <a name="errors-members-dao"></a>Errors Members (DAO)


**Betrifft**: Access 2013, Office 2013

Eine Errors-Auflistung enthält alle gespeicherten Error-Objekte, die sich jeweils auf eine einzelne DAO-Operation beziehen.
