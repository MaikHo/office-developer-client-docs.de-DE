# <a name="step-1-set-up-the-visual-basic-project"></a>Schritt 1: Einrichten des Visual Basic-Projekts


**Betrifft**: Access 2013, Office 2013

## <a name="step-1-set-up-the-visual-basic-project"></a>Schritt 1: Einrichten des Visual Basic-Projekts

Dieses Szenario geht davon aus, dass Sie Microsoft Visual Basic 6.0 oder höher, ADO 2.5 oder höher und den Microsoft OLE DB-Anbieter für Internet Publishing auf dem System installiert haben.

**So erstellen Sie ein ADO-Projekt**

1.  Erstellen Sie in Microsoft Visual Basic ein neues Standard EXE-Projekt.

2.  Klicken Sie im Menü **Project** auf **References**.

3.  Wählen Sie **Microsoft ActiveX Data Objects 2.5 Library** aus, und klicken Sie auf **OK**.

**So fügen Sie Steuerelemente im Hauptformular ein**

1.  Fügen Sie Form1 ein ListBox-Steuerelement hinzu. Legen Sie seine Name-Eigenschaft auf IstMain fest.

2.  Fügen Sie Form1 ein weiteres ListBox-Steuerelement hinzu. Legen Sie seine Name-Eigenschaft auf IstDetails fest.

3.  Fügen Sie Form1 ein TextBox-Steuerelement hinzu. Legen Sie seine Name-Eigenschaft auf IstDetails fest.
