# <a name="indexfields-property-dao"></a>Index.Fields-Eigenschaft (DAO)


**Betrifft**: Access 2013, Office 2013

Gibt eine **Fields**-Auflistung zurück, die alle gespeicherten **Field**-Objekte für das angegebene Objekt enthält. Lese-/Schreibzugriff.

## <a name="syntax"></a>Syntax

*Ausdruck* . Felder

*Ausdruck* Eine Variable, die ein **Index** -Objekt darstellt.
