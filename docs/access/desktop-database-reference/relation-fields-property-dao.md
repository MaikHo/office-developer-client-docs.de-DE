# <a name="relationfields-property-dao"></a>Relation. Fields-Eigenschaft (DAO)


**Gilt für**: Access 2013, Office 2013

Gibt eine **Fields** -Auflistung zurück, die alle gespeicherten **Field** -Objekte für das angegebene Objekt enthält. Schreibgeschützt.

## <a name="syntax"></a>Syntax

*Ausdruck* . Felder

*Ausdruck* Eine Variable, die ein **Relation** -Objekt darstellt.
