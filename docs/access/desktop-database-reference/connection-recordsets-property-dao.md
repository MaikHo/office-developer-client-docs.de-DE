# <a name="connectionrecordsets-property-dao"></a>Connection. Recordsets-Eigenschaft (DAO)


**Gilt für**: Access 2013, Office 2013

Gibt eine **Recordsets**-Auflistung zurück, die alle geöffneten Recordsets für die angegebene Verbindung enthält. Schreibgeschützt.

## <a name="syntax"></a>Syntax

*Ausdruck* . Recordsets

*Ausdruck* Eine Variable, die ein **Connection** -Objekt darstellt.
