# <a name="documentname-property-dao"></a>Document.Name-Eigenschaft (DAO)


**Betrifft**: Access 2013, Office 2013

Gibt den Namen des angegebenen Objekts zurück. Schreibgeschützter **String**-Wert.

## <a name="syntax"></a>Syntax

*Ausdruck* . Name

*Ausdruck* Eine Variable, die ein **Document** -Objekt darstellt.
