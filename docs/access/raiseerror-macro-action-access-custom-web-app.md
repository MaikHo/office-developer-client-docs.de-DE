# <a name="raiseerror-macro-action-access-custom-web-app"></a>Auslösenfehler-Makroaktion (Access benutzerdefinierte Web app)

Die **Auslösenfehler** -Aktion zeigt ein Popupfenster, das eine angegebene Fehlermeldung enthält. 
  
> [!IMPORTANT]
> [!WICHTIG] Das Erstellen und Verwenden von Access-Web-Apps in SharePoint wird von Microsoft nicht mehr empfohlen. Alternativ sollten Sie die Verwendung von [Microsoft PowerApps](https://powerapps.microsoft.com/en-us/) für das Erstellen von Business Solutions ohne Code für das Web und für mobile Geräte in Betracht ziehen. 
  
> [!NOTE]
> [!HINWEIS] Die **AuslösenFehler** -Aktion ist nur in Datenmakros verfügbar. 
  
## <a name="setting"></a>Einstellung

Die **Auslösenfehler** -Aktion hat das folgende Argument. 
  
|**Argument**|**Beschreibung**|
|:-----|:-----|
| _Beschreibung des Fehlers_ <br/> |Ein Zeichenfolgenausdruck, der den Fehler beschreibt.  <br/> |
   
## <a name="remarks"></a>Hinweise

Wenn die **Auslösenfehler** -Aktion aufgerufen wird, werden alle Vorgänge in die aktuelle Transaktion rückgängig gemacht.
