# <a name="-string-concatenation-operator-access-custom-web-app"></a>+ (String-Operator für das Verketten) (Access benutzerdefinierte Web app)

Das Pluszeichen (+) wird der Operator, der zeichenfolgenverkettung ermöglicht. Alle anderen Zeichenfolgen erfolgt mithilfe von Zeichenfolgenfunktionen wie **Teilzeichenfolge**. 
  
> [!IMPORTANT]
> [!WICHTIG] Das Erstellen und Verwenden von Access-Web-Apps in SharePoint wird von Microsoft nicht mehr empfohlen. Alternativ sollten Sie die Verwendung von [Microsoft PowerApps](https://powerapps.microsoft.com/en-us/) für das Erstellen von Business Solutions ohne Code für das Web und für mobile Geräte in Betracht ziehen. 
  
## <a name="syntax"></a>Syntax

 *TextExpression* **+** *TextExpression* 
  
 *TextExpression*  Ein beliebiger Ausdruck gültigen Text.
