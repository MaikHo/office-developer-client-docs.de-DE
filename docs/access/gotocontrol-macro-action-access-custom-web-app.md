# <a name="gotocontrol-macro-action-access-custom-web-app"></a>GeheZuSteuerelement-Makroaktion (Access benutzerdefinierte Web app)

Die **GeheZuSteuerelement** -Aktion können Sie den Fokus auf das angegebene Steuerelement im aktuellen Datensatz der geöffneten Ansicht. 
  
> [!IMPORTANT]
> [!WICHTIG] Das Erstellen und Verwenden von Access-Web-Apps in SharePoint wird von Microsoft nicht mehr empfohlen. Alternativ sollten Sie die Verwendung von [Microsoft PowerApps](https://powerapps.microsoft.com/en-us/) für das Erstellen von Business Solutions ohne Code für das Web und für mobile Geräte in Betracht ziehen. 
  
## <a name="setting"></a>Einstellung

Die **GeheZuSteuerelement** -Aktion hat das folgende Argument. 
  
|**Aktionsargument**|**Beschreibung**|
|:-----|:-----|
|**Steuerelementname** <br/> |Der Name des Felds oder Steuerelement den Fokus werden soll. Dabei handelt es sich um ein Pflichtargument.  <br/> |
   
## <a name="remarks"></a>Hinweise

Sie können diese Aktion verwenden, wenn Sie ein bestimmtes Feld oder Steuerelement den Fokus aufweisen soll. Diese Aktion können auch um in einem Formular gemäß bestimmter Bedingungen zu navigieren. Beispielsweise, wenn der Benutzer in einem Steuerelement Verheiratet No eingibt, kann der Fokus automatisch das Steuerelement Partner/in überspringen und zum nächsten Steuerelement zu verschieben.
