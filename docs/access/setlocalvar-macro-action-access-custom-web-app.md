# <a name="setlocalvar-macro-action-access-custom-web-app"></a>FestlegenLokaleVar-Makroaktion (Access benutzerdefinierte Web app)

Mit der **FestlegenLokaleVar** -Aktion erstellen Sie eine temporäre Variable und legen diese auf einen bestimmten Wert fest. 
  
> [!IMPORTANT]
> [!WICHTIG] Das Erstellen und Verwenden von Access-Web-Apps in SharePoint wird von Microsoft nicht mehr empfohlen. Alternativ sollten Sie die Verwendung von [Microsoft PowerApps](https://powerapps.microsoft.com/en-us/) für das Erstellen von Business Solutions ohne Code für das Web und für mobile Geräte in Betracht ziehen. 
  
> [!NOTE]
> Die **FestlegenLokaleVar** -Aktion ist nur in Datenmakros verfügbar. 
  
## <a name="setting"></a>Einstellung

Die **FestlegenLokaleVar** -Aktion kann mit den folgenden Argumenten verwendet werden. 
  
|**Argumentname**|**Erforderlich**|**Beschreibung**|
|:-----|:-----|:-----|
|**Name** <br/> |Ja  <br/> |Eine Zeichenfolge, die den Namen der Variablen angibt.  <br/> |
|**Expression** <br/> |Ja  <br/> |Ein Ausdruck, der verwendet wird, um den Wert für diese temporäre Variable festzulegen. Setzen Sie den Ausdruck mit dem Gleichheitszeichen (=). Sie können klicken Sie auf die Schaltfläche **Erstellen** , um den **Ausdrucks-Generator** verwenden, um dieses Argument festzulegen.  <br/> |
