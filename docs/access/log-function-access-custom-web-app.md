# <a name="log-function-access-custom-web-app"></a>Log-Funktion (Access Custom Web App)

Gibt den natürlichen Logarithmus oder den Logarithmus der gegebenen Basis des angegebenen Ausdrucks zurück.
  
> [!IMPORTANT]
> Das Erstellen und Verwenden von Access-Web-Apps in SharePoint wird von Microsoft nicht mehr empfohlen. Alternativ sollten Sie die Verwendung von [Microsoft PowerApps](https://powerapps.microsoft.com/en-us/) für das Erstellen von Business Solutions ohne Code für das Web und für mobile Geräte in Betracht ziehen. 
  
## <a name="syntax"></a>Syntax

 **Protokoll** (*Numerischer Ausdruck* , [ *Base* ]) 
  
Die **Log** -Funktion enthält die folgenden Argumente. 
  
|**Argumentname**|**Beschreibung**|
|:-----|:-----|
| *Numerischer Ausdruck*  <br/> |Die positive Zahl, für die der Logarithmus gewünscht wird.  <br/> |
| *Base*  <br/> |Die Basis des Logarithmus. Wenn kein Wert angegeben wird, gibt die **Log** -Funktion den natürlichen Logarithmus zurück.  <br/> |
   
## <a name="remarks"></a>Bemerkungen

Die Funktion **log10** ist ähnlich, gibt jedoch immer den allgemeinen Logarithmus zurück, was bedeutet, dass der Logarithmus für die Basis 10 ist. 
  
Der natürliche Logarithmus ist der Logarithmus zur Basis e, wobei e eine irrationale Konstante ungefähr 2,718281828 ist.
