# <a name="editrecord-macro-action-access-custom-web-app"></a>BearbeitenDatensatz-Makroaktion (Access benutzerdefinierte Web app)

Fügt den aktuellen Datensatz in den Bearbeitungsmodus.
  
> [!IMPORTANT]
> [!WICHTIG] Das Erstellen und Verwenden von Access-Web-Apps in SharePoint wird von Microsoft nicht mehr empfohlen. Alternativ sollten Sie die Verwendung von [Microsoft PowerApps](https://powerapps.microsoft.com/en-us/) für das Erstellen von Business Solutions ohne Code für das Web und für mobile Geräte in Betracht ziehen. 
  
## <a name="syntax"></a>Syntax

 **BearbeitenDatensatz**
  
## <a name="remarks"></a>Hinweise

Verwenden Sie die **SaveRecord** -Aktion zum Speichern von Änderungen an den Datensatz oder die **UndoRecord** -Aktion, um die Änderungen zu verwerfen. 
  
Die **BearbeitenDatensatz** -Aktion ist gleichbedeutend mit der Schaltfläche **Bearbeiten** auf der Aktionsleiste.
