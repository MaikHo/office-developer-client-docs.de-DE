# <a name="reserved-keywords-access-custom-web-app"></a>ReServierte Schlüsselwörter (Access Custom Web App)

Microsoft Access verwendet reservierte Schlüsselwörter zum definieren, bearbeiten und Zugreifen auf Datenbanken. ReServierte Schlüsselwörter sind Teil der Grammatik der Sprache, die von Access zum Analysieren und verstehen von SQL-Anweisungen verwendet wird. 
  
> [!IMPORTANT]
> Das Erstellen und Verwenden von Access-Web-Apps in SharePoint wird von Microsoft nicht mehr empfohlen. Alternativ sollten Sie die Verwendung von [Microsoft PowerApps](https://powerapps.microsoft.com/en-us/) für das Erstellen von Business Solutions ohne Code für das Web und für mobile Geräte in Betracht ziehen.
