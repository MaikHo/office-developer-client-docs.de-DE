# <a name="reserved-keywords-access-custom-web-app"></a>Reservierte Schlüsselwörter (Access benutzerdefinierte Web app)

Microsoft Access verwendet reservierte Schlüsselwörter für definieren, bearbeiten und den Zugriff auf Datenbanken. Reservierte Schlüsselwörter sind Teil der Grammatik der Sprache, die von Access zum Analysieren und Grundlegendes zu SQL-Anweisungen verwendet wird. 
  
> [!IMPORTANT]
> [!WICHTIG] Das Erstellen und Verwenden von Access-Web-Apps in SharePoint wird von Microsoft nicht mehr empfohlen. Alternativ sollten Sie die Verwendung von [Microsoft PowerApps](https://powerapps.microsoft.com/en-us/) für das Erstellen von Business Solutions ohne Code für das Web und für mobile Geräte in Betracht ziehen.
