# <a name="exp-function-access-custom-web-app"></a>EXP-Funktion (Access benutzerdefinierte Web app)

Gibt den exponentiellen Wert des angegebenen Ausdrucks.
  
> [!IMPORTANT]
> [!WICHTIG] Das Erstellen und Verwenden von Access-Web-Apps in SharePoint wird von Microsoft nicht mehr empfohlen. Alternativ sollten Sie die Verwendung von [Microsoft PowerApps](https://powerapps.microsoft.com/en-us/) für das Erstellen von Business Solutions ohne Code für das Web und für mobile Geräte in Betracht ziehen. 
  
## <a name="syntax"></a>Syntax

 **EXP** (*NumericExpression*) 
  
Die **Exp** -Funktion enthält das folgende Argument. 
  
|**Argumentname**|**Description**|
|:-----|:-----|
| *NumericExpression*  <br/> |Ein Ausdruck vom Typ Double-Wert oder einen Typ, der implizit in Double-Wert konvertiert werden kann.  <br/> |
   
## <a name="remarks"></a>Hinweise

Die Konstante **e** (.. 2.718281) ist die Basis natürlicher Logarithmen. 
  
Der Exponent einer Zahl ist die Konstante **e** potenziert mit der Zahl. Beispielsweise **Exp** (1.0) = e ^ 1,0 = 2.71828182845905 und **Exp** (10) = e ^ 10 = 22026.4657948067. 
  
Ist die Anzahl der exponentielle den natürlichen Logarithmus einer Zahl selbst: **Exp** (LOG (n)) = n. Und den natürlichen Logarithmus von den exponentiellen Wert einer Zahl ist die Zahl selbst: LOG (**Exp** (n)) = n.
