# <a name="right-function-access-custom-web-app"></a>Right-Funktion (Access Custom Web App)

Gibt den rechten Teil einer Zeichenfolge mit der angegebenen Anzahl von Zeichen zurück.
  
> [!IMPORTANT]
> Das Erstellen und Verwenden von Access-Web-Apps in SharePoint wird von Microsoft nicht mehr empfohlen. Alternativ sollten Sie die Verwendung von [Microsoft PowerApps](https://powerapps.microsoft.com/en-us/) für das Erstellen von Business Solutions ohne Code für das Web und für mobile Geräte in Betracht ziehen. 
  
## <a name="syntax"></a>Syntax

 **Rechts** (*Text*, *Länge*) 
  
Die **right** -Funktion enthält die folgenden Argumente. 
  
|**Argumentname**|**Beschreibung**|
|:-----|:-----|
| *Text*  <br/> |Ein Textausdruck.  <br/> |
| *Length*  <br/> |Ist eine positive ganze Zahl, die angibt, wie viele Zeichen des *Texts* zurückgegeben werden.  <br/> |
