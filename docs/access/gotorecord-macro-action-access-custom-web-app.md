# <a name="gotorecord-macro-action-access-custom-web-app"></a>GeheZuDatensatz-Makroaktion (Access benutzerdefinierte Web app)

Die **GeheZuDatensatz** -Aktion können Sie den angegebenen Datensatz den aktuellen Datensatz in einer Ansicht machen. 
  
> [!IMPORTANT]
> [!WICHTIG] Das Erstellen und Verwenden von Access-Web-Apps in SharePoint wird von Microsoft nicht mehr empfohlen. Alternativ sollten Sie die Verwendung von [Microsoft PowerApps](https://powerapps.microsoft.com/en-us/) für das Erstellen von Business Solutions ohne Code für das Web und für mobile Geräte in Betracht ziehen. 
  
## <a name="setting"></a>Einstellung

Die **GeheZuDatensatz** -Aktion hat die folgenden Argumente. 
  
|**Aktionsargument**|**Beschreibung**|
|:-----|:-----|
|**Record** <br/> |Der Datensatz zum aktuellen Datensatz machen. Wählen Sie die **vorherigen**, **nächsten**, **ersten**oder **letzten**. Der Standardwert ist **Weiter**.  <br/> |
