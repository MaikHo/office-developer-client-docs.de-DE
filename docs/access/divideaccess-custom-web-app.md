# <a name="-divide-access-custom-web-app"></a>/(Divide) (Access Custom Web App)

Dividiert eine Zahl durch eine andere.
  
> [!IMPORTANT]
> Das Erstellen und Verwenden von Access-Web-Apps in SharePoint wird von Microsoft nicht mehr empfohlen. Alternativ sollten Sie die Verwendung von [Microsoft PowerApps](https://powerapps.microsoft.com/en-us/) für das Erstellen von Business Solutions ohne Code für das Web und für mobile Geräte in Betracht ziehen. 
  
## <a name="syntax"></a>Syntax

 *Dividenden*  /  *Divisor* 
  
 *Dividende*  Ist der numerische Ausdruck, der geteilt werden soll. Kann ein beliebiger gültiger Ausdruck eines beliebigen Datentypen der numerischen Datentypkategorie sein, mit Ausnahme des datetime-Datentyps. 
  
 *Divisor*  Ist der numerische Ausdruck, nach dem die Dividende geteilt werden soll. Kann ein beliebiger gültiger Ausdruck eines beliebigen Datentypen der numerischen Datentypkategorie sein, mit Ausnahme des datetime-Datentyps. 
  
## <a name="return-type"></a>Rückgabetyp

Gibt den Datentyp des Arguments mit der höheren Rangfolge zurück. 
  
Wenn eine ganz ** Zahl Dividende durch einen ganzzahligen *Divisor* geteilt wird, ist das Ergebnis eine ganze Zahl, deren Bruchteil des Ergebnisses abgeschnitten ist. 
  
## <a name="remarks"></a>Bemerkungen

Der vom/-Operator zurückgegebene tatsächliche Wert ist der Quotient des ersten Ausdrucks, dividiert durch den zweiten Ausdruck.
