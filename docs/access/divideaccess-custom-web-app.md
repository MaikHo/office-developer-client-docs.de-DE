# <a name="-divide-access-custom-web-app"></a>/ (Division) (Access benutzerdefinierte Web app)

Dividiert eine Zahl von einem anderen.
  
> [!IMPORTANT]
> [!WICHTIG] Das Erstellen und Verwenden von Access-Web-Apps in SharePoint wird von Microsoft nicht mehr empfohlen. Alternativ sollten Sie die Verwendung von [Microsoft PowerApps](https://powerapps.microsoft.com/en-us/) für das Erstellen von Business Solutions ohne Code für das Web und für mobile Geräte in Betracht ziehen. 
  
## <a name="syntax"></a>Syntax

 *Dividend*  /  *Divisor* 
  
 *Dividend*  Der numerische Ausdruck, geteilt ist. Kann ein beliebiger gültiger Ausdruck des die Datentypen der Typkategorie numerische Daten mit Ausnahme der Datetime-Datentyp. 
  
 *Divisor*  Ist der numerische Ausdruck, um den Teilen der Dividend. Kann ein beliebiger gültiger Ausdruck des die Datentypen der Typkategorie numerische Daten mit Ausnahme der Datetime-Datentyp. 
  
## <a name="return-type"></a>Rückgabetyp

Gibt den Datentyp des Arguments mit höherer Rangfolge zurück. 
  
Ist eine ganze Zahl *Dividend* durch eine ganze Zahl *Divisor* unterteilt, ist das Ergebnis eine ganze Zahl, die jeder Bruchteil des Ergebnisses abgeschnitten hat. 
  
## <a name="remarks"></a>Hinweise

Der tatsächliche Rückgabewert der / Operator ist der Quotient aus dem ersten Ausdruck geteilt durch den zweiten Ausdruck.
