# <a name="guidelines-for-improving-the-performance-of-infopath-forms"></a>Richtlinien zur Leistungsoptimierung von InfoPath-Formularen

Entwerfen von InfoPath-Formularvorlage für eine bessere Leistung, müssen Sie darauf achten, mit der Größe und Anzahl der Ansichten, die Größe und die Struktur des XML- und der Menge der Datei und Bild-Steuerelemente, die Sie verwenden. Sie müssen auch vorsichtig sein, wie Sie Ihre digitalen Signaturen, strukturieren und die Größe und Anzahl von datenverbindungen in das Formular. Befolgen Sie die Richtlinien und bewährten Methoden im Artikel Verbessern der Leistung von InfoPath 2007-Formularen, können Sie Leistungsprobleme beim Entwerfen einer Formularvorlage vermeiden und Problembehandlung in vorhandenen Formularvorlagen.
