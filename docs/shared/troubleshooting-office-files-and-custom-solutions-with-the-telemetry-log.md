<span data-ttu-id="6d4c7-p105">Wenn eine Office-Datei oder -Lösung geladen, verwendet oder geschlossen wird oder einen Fehler in einer der ausgewählten Office 2013-Anwendungen auslöst, fügt die Anwendung einen Datensatz in einem lokalen Datenspeicher (einer Datenbank auf demselben Computer) hinzu, der Informationen über das Ereignis enthält. Der Datensatz umfasst einen Titel für das Ereignis, den Namen der Anwendung, die das Ereignis protokolliert hat, die Uhrzeit, den Namen der Datei oder Lösung, den Schweregrad und eine kurze Beschreibung eventueller Fehler, die aufgetreten sind.
