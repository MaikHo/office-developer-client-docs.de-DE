# <a name="essential-and-useful-c-api-xlm-functions"></a>Wichtige und nützliche C-API XLM-Funktionen

 **Gilt für**: Excel 2013 | Office 2013 | Visual Studio 
  
In diesem Abschnitt beschriebenen Funktionen handelt es sich um Microsoft Excel Callback-Funktionen, die besonders nützlich, um die DLL und XLL-Entwickler sind. Dieser ist die Funktion **XlfRegister** unverzichtbar für XLLs und DLLs, die zu ihren Funktionen und Befehle zu registrieren, sodass sie direkt in Excel aufgerufen werden können. Die Funktionen **XlfUnregister** und **XlfSetName** in Kombination dienen zum Aufheben der Registrierung DLL und XLL-Funktionen und Befehle. 
  
Viele weitere Funktionen werden von Excel über die C-API, die bei der Entwicklung von XLLs nützlich sind verfügbar gemacht werden. Excel-Arbeitsblatt entsprechen Funktionen und die Funktionen und Befehle, die von XLM-Makrovorlagen verfügbar sind.
