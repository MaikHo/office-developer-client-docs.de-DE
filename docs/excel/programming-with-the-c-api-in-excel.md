## <a name="asynchronous-calling-of-excel"></a><span data-ttu-id="92224-154">Asynchrones Aufrufen von Excel</span><span class="sxs-lookup"><span data-stu-id="92224-154">Asynchronous Calling of Excel</span></span>

<span data-ttu-id="92224-p117">Sie erreichen dies, indem Sie Excel aktiv Änderungen abfragen lassen, dies ist ineffizient ist und einschränkend, da Excel dabei häufig seine normale Aktivität unterbrechen muss. <span data-ttu-id="92224-p122">Abgesehen davon, dass die Tabellenblattfunktionen und Makrovorlagenfunktionen und Befehle bereitgestellt werden, liefert die C-API Funktions- und Befehlsaufzählungen, die nur mithilfe dieser Rückrufe innerhalb einer DLL aufgerufen werden können.
