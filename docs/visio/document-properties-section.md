# <a name="document-properties-section"></a>Document Properties Section

Enthält Zellen für ein Dokument, über die die Vorschauqualität, der Geltungsbereich und das Ausgabeformat gesteuert werden.
  
## <a name="remarks"></a>Bemerkungen

 Sie können diese Werte auch mithilfe des Dialogfelds **Eigenschaften** festlegen (klicken Sie auf der Registerkarte **Datei** , klicken Sie auf **Info**, klicken Sie auf **Eigenschaften**, und klicken Sie dann auf **Erweiterte Eigenschaften**).
