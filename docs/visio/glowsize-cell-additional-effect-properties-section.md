# <a name="glowsize-cell-additional-effect-properties-section"></a>GlowSize Cell (Additional Effect Properties Section)

Bestimmt die Größe der der externe Leuchten eines Shapes in Punkt. 
  
## <a name="remarks"></a>Hinweise

Wenn Sie einen Verweis auf die Zelle **GlowSize** nach Namen aus, als Wert des Attributs **N** **ein Zellenelement** , einer anderen Formel oder aus einem Programm mithilfe der **CellsU** -Eigenschaft erhalten möchten, verwenden Sie Folgendes: 
  
|||
|:-----|:-----|
| Zellenname:  <br/> | GlowSize  <br/> |
   
Wenn Sie einen Verweis auf die Zelle **GlowSize** aus einem Programm nach Index erhalten möchten, verwenden Sie die **CellsSRC** -Eigenschaft mit folgenden Argumenten: 
  
|||
|:-----|:-----|
| Abschnittsindex:  <br/> |**Konstanten visSectionObject** <br/> |
| Zeilenindex:  <br/> |**visRowOtherEffectProperties** <br/> |
| Zellenindex:  <br/> |**visGlowSize** <br/> |
