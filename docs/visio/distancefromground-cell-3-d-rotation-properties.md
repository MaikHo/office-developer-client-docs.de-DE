# <a name="distancefromground-cell-3-d-rotation-properties"></a>DistanceFromGround Cell (3-D Rotation Properties)

Legt den Abstand fest, wenn, den das Objekt von Grund auf neu in Punkt, wenn in 3D gedreht ausgelöst wird.
  
## <a name="remarks"></a>Hinweise

Wenn Sie einen Verweis auf die Zelle **DistanceFromGround** nach Namen aus, als Wert des Attributs **N** **ein Zellenelement** , einer anderen Formel oder aus einem Programm mithilfe der **CellsU** -Eigenschaft erhalten möchten, verwenden Sie Folgendes: 
  
|||
|:-----|:-----|
|Zellenname:  <br/> |DistanceFromGround  <br/> |
   
Wenn Sie einen Verweis auf die Zelle **DistanceFromGround** aus einem Programm nach Index erhalten möchten, verwenden Sie die **CellsSRC** -Eigenschaft mit folgenden Argumenten: 
  
|||
|:-----|:-----|
|Abschnittsindex:  <br/> |**Konstanten visSectionObject** <br/> |
|Zeilenindex:  <br/> |**visRow3DRotationProperties** <br/> |
|Zellenindex:  <br/> |**visDistanceFromGround** <br/> |
