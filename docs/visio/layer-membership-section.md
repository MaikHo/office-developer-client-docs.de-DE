## <a name="remarks"></a><span data-ttu-id="f80e5-105">Bemerkungen</span><span class="sxs-lookup"><span data-stu-id="f80e5-105">Remarks</span></span>

<span data-ttu-id="f80e5-p101">Die Layerzuweisung wird als Index zu der Liste der Layer auf dem Zeichenblatt angezeigt. Der Layerindex entspricht der Reihenfolge der Layer im Dialogfeld **Layereigenschaften** (klicken Sie auf der Registerkarte **Start** in der Gruppe **Bearbeiten** auf **Layer**, und klicken Sie dann auf **Layereigenschaften**). Der erste Name in dem Dialogfeld ist Layer 0, der zweite ist Layer 1 und so weiter.</span><span class="sxs-lookup"><span data-stu-id="f80e5-p101">The layer assignment is shown as an index to the list of layers on the page. The layer index corresponds to the order of the layers in the **Layer Properties** dialog box (on the **Home** tab, in the **Editing** group, click **Layers**, and then click **Layer Properties**). The first name in the dialog box is layer 0, the second is layer 1, and so forth.</span></span>
