# <a name="cos-function-visioshapesheet"></a>COS Function (VisioShapeSheet)

Gibt den Kosinus eines Winkels zurück. 
  
## <a name="syntax"></a>Syntax

COS ( *Winkel*) 
  
### <a name="parameters"></a>Parameter

|**Name**|**Erforderlich/Optional**|**Datentyp**|**Beschreibung**|
|:-----|:-----|:-----|:-----|
| _Winkel_ <br/> |Erforderlich  <br/> |**Numerische** <br/> |Der Winkel, dessen Kosinus abgerufen werden soll.  <br/> |
   
## <a name="remarks"></a>Bemerkungen

Die COS-Funktion verwendet Bogenmaß (Radiant), wenn Sie eine andere Maßeinheit in *Winkel* angeben. 
  
## <a name="example"></a>Beispiel

COS(45 Grad) 
  
Gibt 0,7071 zurück.
