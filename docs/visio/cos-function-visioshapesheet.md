# <a name="cos-function-visioshapesheet"></a>COS-Funktion (VisioShapeSheet)

Gibt den Kosinus eines Winkels zurück. 
  
## <a name="syntax"></a>Syntax

COS ( *Winkel*) 
  
### <a name="parameters"></a>Parameter

|**Name**|**Erforderlich/Optional**|**Datentyp**|**Beschreibung**|
|:-----|:-----|:-----|:-----|
| _Winkel_ <br/> |Erforderlich  <br/> |**Numerisch** <br/> |Der Winkel, dessen Kosinus abgerufen werden soll.  <br/> |
   
## <a name="remarks"></a>Bemerkungen

Die COS-Funktion verwendet Radiant, es sei denn, Sie geben eine andere Maßeinheit im *Winkel* an. 
  
## <a name="example"></a>Beispiel

COS(45 Grad) 
  
Gibt 0,7071 zurück.
