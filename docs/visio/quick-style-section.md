# <a name="quick-style-section"></a>Abschnitt "Schnellformatvorlage"

Zeigt die aktuellen Schnellformatvorlagen-Formatierungswerte an, die auf die Form angewendet werden, einschließlich Linienfarbe und-Format, Füllfarbe und-Format, Schattenfarbe, Effekte, Schriftart und Formattyp. 
  
## <a name="remarks"></a>Bemerkungen

Sie können die Form schnell formatieren und dann eine neue Schnellformatvorlage auswählen (Klicken Sie dazu auf der Registerkarte **Start** in der Gruppe **Shape-Formatvorlagen** auf **Weitere**, und wählen Sie dann im Katalog eine andere Schnellformatvorlage aus).
