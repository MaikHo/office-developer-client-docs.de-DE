# <a name="iferror-function"></a>IFERROR Function

Gibt das ausgewertete Ergebnis eines primären-Ausdrucks zurück, wenn es nicht mit einem Fehler ausgewertet wird. Andernfalls wird das ausgewertete Ergebnis eines alternativen Ausdrucks zurückgegeben.
  
## <a name="version-information"></a>Versionsinformationen

Hinzugefügte Version: Visio 2010
 
  
## <a name="syntax"></a>Syntax

IFERROR (** *primärer Ausdruck* **, ** *Alternative Ausdruck* **) 
  
### <a name="parameters"></a>Parameter

|**Name**|**Erforderlich/Optional**|**Datentyp**|**Beschreibung**|
|:-----|:-----|:-----|:-----|
| _primärer Ausdruck_ <br/> |Erforderlich  <br/> |**String** <br/> |Der erste Ausdruck, der ausgewertet werden soll.  <br/> |
| _Alternative Ausdruck_ <br/> |Erforderlich  <br/> |**String** <br/> |Der alternative Ausdruck, der ausgewertet werden soll, wenn der erste Ausdruck einen Fehler zurückgibt.  <br/> |
