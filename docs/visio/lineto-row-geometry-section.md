# <a name="lineto-row-geometry-section"></a>LineTo Row (Geometry Section)

Enthält die *X* - und *y* -Koordinaten des Endscheitelpunkts eines geraden Linienabschnitts. 
  
Eine Zeile LineTo enthält folgende Zellen.
  
|**Cell**|**Beschreibung**|
|:-----|:-----|
|[X](x-cell-geometry-section.md) <br/> |Die *X* -Koordinate des Endscheitelpunkts eines geraden Linienabschnitts.  <br/> |
|[Y](y-cell-geometry-section.md) <br/> |Die *y* -Koordinate des Endscheitelpunkts eines geraden Linienabschnitts.  <br/> |
