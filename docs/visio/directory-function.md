# <a name="directory-function"></a>DIRECTORY Function

Gibt den vollständigen Pfad des Verzeichnisses (Ordner), in dem ein Dokument, als Zeichenfolge gespeichert wird, zurück. Gibt eine leere Zeichenfolge zurück, wenn das Dokument nicht gespeichert wurde.
  
## <a name="syntax"></a>Syntax

DIRECTORY ()
