# <a name="directory-function"></a>DIRECTORY Function

Gibt den vollständigen Pfad des Verzeichnisses (Ordner) zurück, in dem ein Dokument gespeichert wird, als Zeichenfolge. Gibt eine leere Zeichenfolge zurück, wenn das Dokument nicht gespeichert wurde.
  
## <a name="syntax"></a>Syntax

DIRECTORY ()
