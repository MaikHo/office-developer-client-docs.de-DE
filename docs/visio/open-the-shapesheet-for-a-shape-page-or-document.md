# <a name="open-the-shapesheet-for-a-shape-page-or-document"></a>Öffnen des ShapeSheet für ein Shape, eine Seite oder ein Dokument

Das ShapeSheet-Fenster wird in Microsoft Visio standardmäßig ausgeblendet. Zugriff auf das ShapeSheet müssen Sie Visio im Entwicklermodus (Anzeigen der Registerkarte Entwicklertools) auszuführen.
  
### <a name="to-run-visio-in-developer-mode"></a>Ausführen von Visio im Entwicklermodus

1. Klicken Sie auf der Registerkarte **Datei** , und klicken Sie dann auf **Optionen**.
    
2. Klicken Sie im Dialogfeld **Visio-Optionen** klicken Sie auf **Erweitert**, und wählen Sie dann unter **Allgemein** **im Entwicklermodus ausführen**.
    
### <a name="to-view-the-shapesheet-for-a-shape"></a>Anzeigen des ShapeSheet für ein Shape

- Mit der rechten Maustaste in der Form auf Zeichenbereich, und klicken Sie auf **ShapeSheet anzeigen**.
    
- Wählen Sie die Form im Zeichenbereich, und klicken Sie dann auf **Shape**klicken Sie dann auf der Registerkarte **Entwicklertools** in der Gruppe **Shape-Design** auf **ShapeSheet anzeigen**.
    
### <a name="to-view-the-sheet-for-a-page"></a>Anzeigen des Blatts für eine Seite

- Klicken Sie auf der Registerkarte **Entwicklertools** in der Gruppe **Shape-Design** klicken Sie auf **ShapeSheet anzeigen**, und klicken Sie dann auf **Seite**.
    
### <a name="to-view-the-sheet-for-a-document"></a>Anzeigen des Blatts für ein Dokument

- Klicken Sie auf der Registerkarte **Entwicklertools** in der Gruppe **Shape-Design** klicken Sie auf **ShapeSheet anzeigen**, und klicken Sie dann auf **Dokument**.
