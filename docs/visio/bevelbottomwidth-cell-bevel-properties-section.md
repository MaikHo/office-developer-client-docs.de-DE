# <a name="bevelbottomwidth-cell-bevel-properties-section"></a>BevelBottomWidth Cell (Bevel Properties Section)

Legt die Breite der unteren Abschrägung in Punkt. 
  
## <a name="remarks"></a>Bemerkungen

Wenn Sie einen Verweis auf die Zelle **BevelBottomWidth** nach Namen aus, als Wert des Attributs **N** **ein Zellenelement** , einer anderen Formel oder aus einem Programm mithilfe der **CellsU** -Eigenschaft erhalten möchten, verwenden Sie Folgendes: 
  
|||
|:-----|:-----|
| Zellenname:  <br/> | BevelBottomWidth  <br/> |
   
Wenn Sie einen Verweis auf die Zelle **BevelBottomWidth** aus einem Programm nach Index erhalten möchten, verwenden Sie die **CellsSRC** -Eigenschaft mit folgenden Argumenten: 
  
|||
|:-----|:-----|
| Abschnittsindex:  <br/> |**Konstanten visSectionObject** <br/> |
| Zeilenindex:  <br/> |**visRowBevelProperties** <br/> |
| Zellenindex:  <br/> |**visBevelBottomWidth** <br/> |
