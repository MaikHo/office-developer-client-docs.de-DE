# <a name="bevelbottomwidth-cell-bevel-properties-section"></a>Zelle "BevelBottomWidth" (Abschnitt "Fase Properties")

Bestimmt die Breite der unteren Abschrägung in Punkt. 
  
## <a name="remarks"></a>Bemerkungen

Wenn Sie einen Verweis auf die Zelle **BevelBottomWidth** aus einer anderen Formel, nach dem Wert des **N** -Attributs eines **Cell** -Elements oder aus einem Programm mithilfe der **CellsU** -Eigenschaft nach Namen erhalten möchten, verwenden Sie Folgendes: 
  
|||
|:-----|:-----|
| Zellenname:  <br/> | BevelBottomWidth  <br/> |
   
Wenn Sie einen Verweis auf die Zelle **BevelBottomWidth** aus einem Programm nach Index erhalten möchten, verwenden Sie die **CellsSRC** -Eigenschaft mit folgenden Argumenten: 
  
|||
|:-----|:-----|
| Abschnittsindex:  <br/> |**Konstanten visSectionObject** <br/> |
| Zeilenindex:  <br/> |**visRowBevelProperties** <br/> |
| Zellenindex:  <br/> |**visBevelBottomWidth** <br/> |
