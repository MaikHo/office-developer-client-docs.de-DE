# <a name="cell-element-line-gradient-section-visio-xml"></a>Zellenelement (Abschnitt "Linienverlauf") (' Visio XML ')

Enthält die Farbe, Transparenz oder Position eines Farbverlaufsstopps für einen Linien Farbverlauf.
  
## <a name="element-information"></a>Informationen zum Element

|||
|:-----|:-----|
|**Elementtyp** <br/> |[Cell_Type](cell_type-complextypevisio-xml.md) <br/> |
|**Namespace** <br/> |https://schemas.microsoft.com/office/visio/2012/main  <br/> |
|**Schemadatei** <br/> |VisioSchema15. xsd  <br/> |
|**Dokumentteile** <br/> |Document. XML, Master #. XML, Page #. XML  <br/> |
   
## <a name="definition"></a>Definition

```XML
< xs:element name="Cell"  type="Cell_Type" minOccurs="0" maxOccurs="unbounded">
</xs:element >
```

Das **N** -Attribut dieses **Cell** -Elements muss einer einer begrenzten Menge von Werten sein, die ShapeSheet-Zellen entsprechen. In der nachstehenden Tabelle finden Sie die Werte des **N** -Attributs, die für dieses **Cell** -Element zulässig sind. 
  
|**Wert**|**Beschreibung**|**Weitere Informationen**|
|:-----|:-----|:-----|
|GradientStopColor  <br/> |Der Farbwert des Farbverlaufsstopps.  <br/> |[Verlaufs Unterbrechungs Zeile (Abschnitt "Linienverlauf")](gradient-stop-row-line-gradient-section.md) <br/> |
|GradientStopColorTrans  <br/> |Der Transparenzgrad der Farbverlaufs Unterbrechung als Prozentsatz.  <br/> |[Verlaufs Unterbrechungs Zeile (Abschnitt "Linienverlauf")](gradient-stop-row-line-gradient-section.md) <br/> |
|GradientStopPosition  <br/> |Die Position des Farbverlaufsstopps entlang der Richtung des Linien Verlaufs, als Prozentsatz vom Ursprungspunkt des Farbverlaufs bis zur äußeren Kante des Farbverlaufs.  <br/> |[Verlaufs Unterbrechungs Zeile (Abschnitt "Linienverlauf")](gradient-stop-row-line-gradient-section.md) <br/> |
