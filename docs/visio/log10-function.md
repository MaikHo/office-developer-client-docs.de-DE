# <a name="log10-function"></a>LOG10 Function

Gibt den Logarithmus zur Basis 10 einer Zahl zurück. Die Zahl positiv sein muss, oder gibt LOG10 den Fehlerwert # ZAHL.
  
## <a name="syntax"></a>Syntax

LOG10 (** *Anzahl* **) 
  
### <a name="parameters"></a>Parameter

|**Name**|**Erforderlich/Optional**|**Datentyp**|**Beschreibung**|
|:-----|:-----|:-----|:-----|
| _Anzahl_ <br/> |Erforderlich  <br/> |**Numerische** <br/> | Die Zahl, deren Logarithmus zur Basis 10 ermittelt werden soll.  <br/> |
   
### <a name="return-value"></a>R�ckgabewert

Numeric
  
## <a name="example"></a>Beispiel

LOG10(42) 
  
Gibt 1,6232 zurück.
