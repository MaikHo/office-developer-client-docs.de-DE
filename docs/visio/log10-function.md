# <a name="log10-function"></a>LOG10 Function

Gibt den Logarithmus einer Zahl für Basis 10 zurück. Die Zahl muss positiv sein, oder LOG10 gibt den Fehlerwert #NUM zurück.
  
## <a name="syntax"></a>Syntax

LOG10 (* * *Number* * *) 
  
### <a name="parameters"></a>Parameter

|**Name**|**Erforderlich/Optional**|**Datentyp**|**Beschreibung**|
|:-----|:-----|:-----|:-----|
| _Anzahl_ <br/> |Erforderlich  <br/> |**Numerisch** <br/> | Die Zahl, deren Logarithmus zur Basis 10 ermittelt werden soll.  <br/> |
   
### <a name="return-value"></a>Rückgabewert

Numerisch
  
## <a name="example"></a>Beispiel

LOG10 (42) 
  
Gibt 1,6232 zurück.
