# <a name="not-function"></a>NOT Function

Gibt TRUE (1) zurück __ , wenn logicIVE auf FALSE festgelegt ist. Andernfalls wird FALSE (0) zurückgegeben. 
  
## <a name="syntax"></a>Syntax

NOT (* * *logischer* * *) 
  
### <a name="parameters"></a>Parameter

|**Name**|**Erforderlich/Optional**|**Datentyp**|**Beschreibung**|
|:-----|:-----|:-----|:-----|
| _Logik_ <br/> |Erforderlich  <br/> |**String** <br/> |Der logische Ausdruck, der ausgewertet werden soll.  <br/> |
   
### <a name="return-value"></a>Rückgabewert

Boolesch
  
## <a name="example"></a>Beispiel

NICHT (Höhe \> 0,75 in) 
  
Gibt 1 zurück, wenn Höhe kleiner als oder gleich 2 cm ist. Gibt 0 zurück, wenn Höhe größer als 2 cm ist.
