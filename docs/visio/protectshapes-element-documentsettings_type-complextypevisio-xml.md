# <a name="protectshapes-element-documentsettingstype-complextype-visio-xml"></a>ProtectShapes-Element (DocumentSettings_Type ComplexType) ("Visio XML")

Gibt an, ob der Benutzer für die Auswahl von Shapes, die ihre **LockSelect** -Element, das auf 1 festgelegt haben verhindert wird.

|**Schemadatei** <br/> |VisioSchema15.xsd  <br/> |
|**Dokumentbausteine** <br/> |Document.Xml  <br/> |

```XML
< xs:element name="ProtectShapes" type="ProtectShapes_Type" minOccurs="0" maxOccurs="1" >
</xs:element >
```
