# <a name="text-block-format-section"></a>Text Block Format Section

Enthält Zellen zum Steuern der Ausrichtung, der Ränder und der Standardtabstoppposition von Text im Textblock eines Shapes.
  
## <a name="remarks"></a>Bemerkungen

Sie können diese Textattribute auch festlegen, indem Sie das Dialogfeld **Text** verwenden (klicken Sie auf der Registerkarte **Start** auf das Dialogfeld-Startprogramms in der Gruppe **Schriftart** ).
