# <a name="3-d-rotation-properties-section"></a>3D-Rotations Eigenschaften (Abschnitt)

Zeigt die aktuellen Werte des Drehwinkels einer Form entlang der x-, y-und z-Achsen an.
  
> [!NOTE]
> Der Abschnitt 3D-Rotations Eigenschaften enthält keine Zellen, die die Auswirkungen der Formkanten steuern. Wenn Sie einen Kanteneffekt festlegen möchten, ändern Sie die Werte der Zellen im Abschnitt [abgeflachte Eigenschaften](bevel-properties-section.md) . 
  
## <a name="remarks"></a>Bemerkungen

Sie können die 3D-Rotationswerte im Dialogfeld **Shape formatieren** ändern (**** > **3D** -Drehungen der**Start** > -**Shape-Formate** > oder eine Änderung an einer der Zellen in den **3D-Rotations Eigenschaften). **Abschnitt.
