# <a name="enter-an-empty-formula"></a>Eingeben einer leeren Formel

1. Wählen Sie die Zelle aus, die die Formel enthalten soll.
    
2. Drücken Sie auf ENTF und anschließend die EINGABETASTE.
    
> [!NOTE]
> Versuchen Sie dies in einer Zelle **Scratch** nach dem eine lokale Formel einfügen. Die Zelle fällt "No Formula," zurück, da keine Formel zum Vererben des Master-Shapes vorhanden ist.
