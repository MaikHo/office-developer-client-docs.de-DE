# <a name="stencilgrouppos-element-windowtype-complextype-visio-xml"></a>StencilGroupPos-Element (Window_Type ComplexType) ("Visio XML")

|**Schemadatei** <br/> |VisioSchema15.xsd  <br/> |

</xs:element>
