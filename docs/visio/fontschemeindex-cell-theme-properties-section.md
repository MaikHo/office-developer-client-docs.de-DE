# <a name="fontschemeindex-cell-theme-properties-section"></a>FontSchemeIndex Cell (Theme Properties Section

Bestimmt das Schriftartenschema eines Designs, das dem Shape als ganze Zahl angewendet wird. 
  
## <a name="remarks"></a>Hinweise

Wenn Sie einen Verweis auf die Zelle **FontSchemeIndex** nach Namen aus, als Wert des Attributs **N** **ein Zellenelement** , einer anderen Formel oder aus einem Programm mithilfe der **CellsU** -Eigenschaft erhalten möchten, verwenden Sie Folgendes: 
  
|||
|:-----|:-----|
| Zellenname:  <br/> | FontSchemeIndex  <br/> |
   
Wenn Sie einen Verweis auf die Zelle **FontSchemeIndex** aus einem Programm nach Index erhalten möchten, verwenden Sie die **CellsSRC** -Eigenschaft mit folgenden Argumenten: 
  
|||
|:-----|:-----|
| Abschnittsindex:  <br/> |**Konstanten visSectionObject** <br/> |
| Zeilenindex:  <br/> |**visRowThemeProperties** <br/> |
| Zellenindex:  <br/> |**visFontSchemeIndex** <br/> |
