# <a name="fillgradientangle-cell-gradient-properties-section"></a>Zelle "FillGradientAngle" (Abschnitt "Gradient Properties")

Bestimmt den Winkel des Füll Verlaufs für Farbverläufe mit einer linearen Richtung in Grad. 
  
## <a name="remarks"></a>Bemerkungen

Wenn Sie einen Verweis auf die Zelle **FillGradientAngle** aus einer anderen Formel, nach dem Wert des **N** -Attributs eines **Cell** -Elements oder aus einem Programm mithilfe der **CellsU** -Eigenschaft nach Namen erhalten möchten, verwenden Sie Folgendes: 
  
|||
|:-----|:-----|
| Zellenname:  <br/> | FillGradientAngle  <br/> |
   
Wenn Sie einen Verweis auf die Zelle **FillGradientAngle** aus einem Programm nach Index erhalten möchten, verwenden Sie die **CellsSRC** -Eigenschaft mit folgenden Argumenten: 
  
|||
|:-----|:-----|
| Abschnittsindex:  <br/> |**Konstanten visSectionObject** <br/> |
| Zeilenindex:  <br/> |**visRowGradientProperties** <br/> |
| Zellenindex:  <br/> |**visFillGradientAngle** <br/> |
