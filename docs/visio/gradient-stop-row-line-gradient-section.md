# <a name="gradient-stop-row-line-gradient-section"></a>Gradient Stop Row (Line Gradient Section)

Enthält die Farbe, Transparenz und Position des einen Farbverlaufstopp für einen Farbverlauf Linie.
  
Eine Zeile **Farbverlauf beenden** enthält folgende Zellen. 
  
|**Cell**|**Beschreibung**|
|:-----|:-----|
|**Color** <br/> |Der Wert der Farbe des Farbverlaufstopps. Dieser Wert kann als die Indexnummer einer Farbe in der Palette Dokument oder die Funktionen [RGB](rgb-function-visioshapesheet.md), [THEMEVAL](themeval-function.md)oder [HSL](hsl-function.md) (z. b) mit ausgedrückt werden.  <br/> |
|**ColorTrans** <br/> |Der Umfang der Transparenz der Farbe für den Farbverlauf, als Prozentsatz.  <br/> |
|**Position** <br/> |Die Position des Farbverlaufstopps in Richtung des Farbverlaufs Linie als Prozentsatz aus der Ausgangspunkt des Farbverlaufs an den äußeren Rand der Zeile.  <br/> |
