# <a name="beginx-cell-1-d-endpoints-section"></a>Zelle "BeginX" (Abschnitt "1-D Endpoints")

Stellt die *x* -Koordinate des Anfangspunkts des 1D-Shapes im Verhältnis zum Ursprung des übergeordneten Objekts dar. 
  
## <a name="remarks"></a>Bemerkungen

Wenn Sie einen Verweis auf die Zelle Zelle BeginX aus einer anderen Formel oder aus einem Programm mithilfe der **CellsU** -Eigenschaft nach Namen erhalten möchten, verwenden Sie Folgendes: 
  
|||
|:-----|:-----|
| Zellenname:  <br/> | BeginX  <br/> |
   
Wenn Sie einen Verweis auf die Zelle Zelle BeginX aus einem Programm nach Index erhalten möchten, verwenden Sie die **CellsSRC** -Eigenschaft mit folgenden Argumenten: 
  
|||
|:-----|:-----|
| Abschnittsindex:  <br/> |**Konstanten visSectionObject** <br/> |
| Zeilenindex:  <br/> |**visRowXForm1D** <br/> |
| Zeilenindex:  <br/> |**vis1DBeginX** <br/> |
