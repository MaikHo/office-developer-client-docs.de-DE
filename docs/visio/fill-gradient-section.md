# <a name="fill-gradient-section"></a>Abschnitt "Füll Verlauf"

Zeigt die Farbwerte für den Füll Verlauf an, die auf das Shape angewendet werden. 
  
## <a name="remarks"></a>Bemerkungen

Jede Verlaufs Unterbrechungs Zeile [(Füll Verlauf)](gradient-stop-row-fill-gradient-section.md) im Abschnitt stellt einen Farbverlaufsstopp in der Füllung dar. Dieser Abschnitt kann leer sein. 
  
Nur die ersten 10 Zeilen im Abschnitt werden von den Farbverlaufseinstellungen verwendet. Alle **Verlaufs Unterbrechungs** Zeilen jenseits von 10 werden ignoriert. 
  
Sie können die Eigenschaften für den Füll Verlauf im **Shape** -Bereich formatieren (Klicken Sie auf der Registerkarte **Start** in der Gruppe **Shape-Formatvorlagen** auf **Füllung**, und klicken Sie dann auf **Fülloptionen**). 
  
Sie können eine graduelle Füllung für das Shape aktivieren oder deaktivieren, indem Sie den Wert der Zelle [FillGradientEnabled Cell (Gradient Properties section)](fillgradientenabled-cell-gradient-properties-section.md) ändern.
