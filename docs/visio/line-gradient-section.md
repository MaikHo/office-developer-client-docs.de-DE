# <a name="line-gradient-section"></a>Line Gradient Section

Zeigt die RGB-Werte für die Zeile Farbverlauf auf die Form angewendet. 
  
## <a name="remarks"></a>Bemerkungen

Jede Zeile [Farbverlauf beenden Zeile (Zeile Farbverlauf Abschnitt)](gradient-stop-row-line-gradient-section.md) im Abschnitt stellt einen Farbverlaufstopp in der Zeile. In diesem Abschnitt kann leer sein. 
  
Nur die ersten 10 Zeilen im Abschnitt enthalten, werden durch die Einstellungen der Farbverlauf verwendet. Alle **Farbverlauf beenden** Zeilen unterhalb von 10 werden ignoriert. 
  
Sie können die Zeile Farbverlauf Eigenschaften festlegen, in den Bereich **Zeile** im **Shape formatieren** (klicken Sie auf der Registerkarte **Start** in der Gruppe **Formenarten** klicken Sie auf die **Zeile**, und klicken Sie dann auf **Optionen**). 
  
Sie können aktivieren oder deaktivieren eine Farbverlaufstopp Linienfarbe für das Shape durch Ändern des Werts der Zelle [LineGradientEnabled Zelle (Farbverlauf Eigenschaften im Abschnitt)](linegradientenabled-cell-gradient-properties-section.md) .
