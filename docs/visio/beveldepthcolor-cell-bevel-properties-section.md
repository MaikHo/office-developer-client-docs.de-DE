# <a name="beveldepthcolor-cell-bevel-properties-section"></a>Zelle "BevelDepthColor" (Abschnitt "Fase Properties")

Bestimmt die Farbe der tiefen der Abschrägung, als RGB-Wert oder vom aktiven Design abhängig.
  
## <a name="remarks"></a>Bemerkungen

Wenn Sie einen Verweis auf die Zelle **BevelDepthColor** aus einer anderen Formel, nach dem Wert des **N** -Attributs eines **Cell** -Elements oder aus einem Programm mithilfe der **CellsU** -Eigenschaft nach Namen erhalten möchten, verwenden Sie Folgendes: 
  
|||
|:-----|:-----|
| Zellenname:  <br/> | BevelDepthColor  <br/> |
   
Wenn Sie einen Verweis auf die Zelle **BevelDepthColor** aus einem Programm nach Index erhalten möchten, verwenden Sie die **CellsSRC** -Eigenschaft mit folgenden Argumenten: 
  
|||
|:-----|:-----|
| Abschnittsindex:  <br/> |**Konstanten visSectionObject** <br/> |
| Zeilenindex:  <br/> |**visRowBevelProperties** <br/> |
| Zellenindex:  <br/> |**visBevelDepthColor** <br/> |
