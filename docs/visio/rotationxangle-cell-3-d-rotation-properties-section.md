# <a name="rotationxangle-cell-3-d-rotation-properties-section"></a>RotationXAngle Cell (3-D Rotation Properties Section)

Bestimmt den Winkel der Drehung auf der X-Achse in Grad (0,0 - 359.9). 
  
## <a name="remarks"></a>Bemerkungen

Wenn Sie einen Verweis auf die Zelle **RotationXAngle** nach Namen aus, als Wert des Attributs **N** **ein Zellenelement** , einer anderen Formel oder aus einem Programm mithilfe der **CellsU** -Eigenschaft erhalten möchten, verwenden Sie Folgendes: 
  
|||
|:-----|:-----|
|Zellenname:  <br/> |RotationXAngle  <br/> |
   
Wenn Sie einen Verweis auf die Zelle **RotationXAngle** aus einem Programm nach Index erhalten möchten, verwenden Sie die **CellsSRC** -Eigenschaft mit folgenden Argumenten: 
  
|||
|:-----|:-----|
|Abschnittsindex:  <br/> |**Konstanten visSectionObject** <br/> |
|Zeilenindex:  <br/> |**visRow3DRotationProperties** <br/> |
|Zellenindex:  <br/> |**visRotationXAngle** <br/> |
