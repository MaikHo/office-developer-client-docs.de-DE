# <a name="bevellightingtype-cell-bevel-properties-section"></a>BevelLightingType Cell (Bevel Properties Section)

Bestimmt den Typ der Beleuchtung wird von der Abschrägungseffekt verwendet.
  
|**Wert**|**Beschreibung**|
|:-----|:-----|
|0  <br/> |Keine Beleuchtung  <br/> |
|1  <br/> |Drei Lichtpunkte  <br/> |
|2  <br/> |Lastenausgleich  <br/> |
|3  <br/> |Soft  <br/> |
|4  <br/> |Effekt hart  <br/> |
|5  <br/> |Flood  <br/> |
|6  <br/> |Gegenüberstellung der  <br/> |
|7  <br/> |Morgen  <br/> |
|8  <br/> |Sonnenaufgang  <br/> |
|9  <br/> |Sonnenuntergang  <br/> |
|10  <br/> |Effekt Kühl  <br/> |
|11  <br/> |Einfrieren  <br/> |
|12  <br/> |Flach  <br/> |
|13  <br/> |Zwei-Punkt  <br/> |
|14  <br/> |Leuchten  <br/> |
|15  <br/> |Heller Raum  <br/> |
   
## <a name="remarks"></a>Bemerkungen

Wenn Sie einen Verweis auf die Zelle **BevelLightingType** nach Namen aus, als Wert des Attributs **N** **ein Zellenelement** , einer anderen Formel oder aus einem Programm mithilfe der **CellsU** -Eigenschaft erhalten möchten, verwenden Sie Folgendes: 
  
|||
|:-----|:-----|
|Zellenname:  <br/> |BevelLightingType  <br/> |
   
Wenn Sie einen Verweis auf die Zelle **BevelLightingType** aus einem Programm nach Index erhalten möchten, verwenden Sie die **CellsSRC** -Eigenschaft mit folgenden Argumenten: 
  
|||
|:-----|:-----|
|Abschnittsindex:  <br/> |**Konstanten visSectionObject** <br/> |
|Zeilenindex:  <br/> |**visRowBevelProperties** <br/> |
|Zellenindex:  <br/> |**visBevelLightingType** <br/> |
