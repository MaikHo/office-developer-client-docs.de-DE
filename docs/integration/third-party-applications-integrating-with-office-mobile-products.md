# <a name="third-party-applications-integrating-with-office-mobile-products-on-windows-android-and-ios"></a>Integrieren von Office Mobile-Produkte bei Windows, iOS und Android drittanbieteranwendungen
 
Als autorisierter Vertreter des Unternehmens stimme ich zu, die korrekten dateispezifischen Besitzertypinformationen (Einstufung einer Datei als geschäftliche oder persönliche/Verbraucherdatei) über die Expando-Eigenschaften unter Windows 10, Storage Access Framework unter Android und die Dokumentauswahl unter iOS an Microsoft zu übermitteln. Geschäftliche Dateien sind die Dateien, die auf Ihrem für Unternehmen vorgesehenen Speicherangebot gespeichert sind. Persönliche/Verbraucherdateien sind die Dateien, die auf Ihrem für Verbraucher vorgesehenen Speicherangebot gespeichert sind.
  
Ich bestätige weiterhin, dass (a) die nicht ordnungsgemäße Identifikation von Dokumenten als geschäftliche oder Verbraucherdateien dazu führen kann, dass die Möglichkeit meines Unternehmens zur Integration in Microsoft Office-Produkte beendet wird; (b) Microsoft diese Bedingungen von Zeit zu Zeit aktualisieren kann und es mir oder einem anderen in meiner Funktion handelnden Mitarbeiter obliegt zu überprüfen, ob neue Bedingungen gelten, bevor nachfolgende Dateiinformationen übergeben werden. Wenn Microsoft Bedingungen ändert, gibt das Gültigkeitsdatum oben auf dieser Inhaltsseite Aufschluss über die Änderungen, z. B. "Letzte Aktualisierung im Monat / Jahr".
